## Summary

Current `main` at `41c5ffc5d5f4dc0f1df6f2c9a7efd8ce6a627f00` does not implement the proposed rolling context-pruning mode. The embedded-runner gate still only enables pruning for `mode === "cache-ttl"`, and the config type/schema surfaces still only accept `"off" | "cache-ttl"`.
