## Summary

Current `main` already supersedes this PR: the shipped pre-commit hook is now formatting-only, tests explicitly assert that it does not run `pnpm check:changed`, and the helper path no longer depends on a bare `pnpm` invocation.

## Evidence

- **Current pre-commit hook no longer runs pnpm checks:** `git-hooks/pre-commit` only collects staged files, filters format targets, runs `oxfmt`, and restages files. There is no `pnpm check:changed --staged` path left for this PR to patch.
