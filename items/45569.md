## Summary

Keep open. Current `main` still appears to require two Enter presses for `/status` in the web chat slash menu: `/status` is explicitly kept off the local auto-execute path, and the UI only auto-sends slash selections when `executeLocal` is true and the command has no args.

## Evidence
