## Summary

Keep open. Current `main` has landed parts of the reported fix set for dreaming corpus hygiene and REM tag filtering, but the follow-up legacy-tag/light-dream path from the issue comments is still present: existing persisted `conceptTags` are read back without stopword re-filtering and light-dream narrative themes still consume those tags directly.
