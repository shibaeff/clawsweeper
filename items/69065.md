## Summary

Current main still treats managed Gateway service install as supported only on macOS, Linux/systemd, and Windows Scheduled Tasks, while the Android docs scope Android to a companion node that does not host the Gateway. That makes the report look more like a supported-surface/docs/UX mismatch than a safely closable obsolete bug, and there is no strong evidence that current main added Android/Termux service support or fully resolved the requested guidance gap.
