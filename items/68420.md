## Summary

Keep open. Current `main` still surfaces incomplete-turn exits as local user-visible errors instead of rotating to the next configured fallback model. Recent `main` work improved Gemini incomplete-turn recovery, but it did not implement this PR's fallback-path change, so there is no conservative auto-close basis.
