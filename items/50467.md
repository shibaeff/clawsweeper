## Summary

Current `main` already contains targeted fixes for both halves of #50467: isolated cron runs no longer prompt for a summary and explicitly retry acknowledgement-only first turns, and the Feishu channel now preserves reply-target metadata from inbound dispatch through outbound send paths with regression coverage.

The current Feishu docs also gate the supported path to newer builds than the reported `2026.3.11` (`docs/channels/feishu.md:17`).
