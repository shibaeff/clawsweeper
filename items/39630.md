## Summary

Keep open. Current `main` does not expose a `web_fetch` private-network opt-in: the public `tools.web.fetch` schema only allows the narrower `ssrfPolicy.allowRfc2544BenchmarkRange` escape hatch, and the runtime only threads that flag into the fetch guard. The docs and changelog match that shipped behavior. The PR discussion also contains an unresolved security-policy disagreement about hard-coded IMDS denylisting inside private IPv6 space, so there is not strong evidence for a conservative cleanup close.
