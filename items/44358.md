## Summary

Keep open. Current `main` still ships OpenProse syntax that restricts `agent.model` and `session.model` to the aliases `sonnet|opus|haiku`, while the issue reports fully qualified ids like `openai/gpt-5.2`. That means the requested behavior is not implemented on `main` or in the latest release, so there is not strong evidence to close this item.
