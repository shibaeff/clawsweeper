## Summary

Keep open. Current `main` already keeps Mattermost DMs on the base session, but it does not include this PR's reply-context preservation, typing, or websocket liveness changes, and the PR discussion shows the author addressed the main review findings rather than the work becoming obsolete.

## Evidence

That matches the existing docs/tests, but it also means inbound DM reply roots are not carried forward as reply context.
