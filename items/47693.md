## Summary

Keep open. Current `main` still does not contain the Slack DM drop-gate verbose logging from this PR, and the batched `appMentionRetryKeys` cleanup bug described in the PR body is still present. The only recorded review comment also points out a small follow-up needed on the PR branch itself (`shouldLogVerbose()` guard in `prepare.ts`).
