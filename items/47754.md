## Summary

Keep open. Current `main` and the latest shipped release still build keyword-result snippets by truncating the stored chunk text from the beginning, so the reported deep-match visibility bug is not implemented away. The PR’s approach remains relevant, but it now needs a rebase onto the post-refactor `extensions/memory-core` path.
