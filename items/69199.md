## Summary

Keep open. Current `main` still returns the generic embedding/provider warning and action for `memory_search` failures, and the sqlite-specific test proposed by this PR is not present. The repo does have a lower-level `requireNodeSqlite()` error mentioning missing `node:sqlite`, but that message is not translated into the user-facing unavailable payload this PR is trying to fix.
