## Summary

Keep open. Current `main` does not implement this PR's proposed `channels.feishu.localRoots` / `"any"` config surface. Feishu outbound media still only consumes core-provided `mediaLocalRoots`, while the public Feishu config docs and schema do not expose a channel/account `localRoots` option. Main already shipped the narrower `#27884` fix for propagating `mediaLocalRoots`, so this PR is not implemented on `main` but also is not safe to auto-close as obsolete.
