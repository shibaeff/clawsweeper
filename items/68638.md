## Summary

Superseded by current `main`: OpenClaw now keeps non-activating snapshot registration intact while restoring compaction, memory, agent-harness, and detached-task global state after `register()`, which covers the leakage this PR was narrowed to fix.

## Evidence

The author then narrowed the PR to preventing compaction/global leakage while preserving snapshot registration semantics.
