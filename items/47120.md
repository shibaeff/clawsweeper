## Summary

Keep open. Current `main` has backend support for derived titles and last-message previews, and the TUI uses it, but the Control UI sessions view still does not request or render that data.

## Evidence

- **Control UI list request omits preview/title flags:** `loadSessions()` builds `sessions.list` params with only `includeGlobal`, `includeUnknown`, `activeMinutes`, and `limit`; it never asks for `includeDerivedTitles` or `includeLastMessage`.

  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **Control UI sessions table has no preview column or combined label:** The sessions table renders columns for Key, Label, Kind, Updated, Tokens, and config fields only. In the row renderer, the key cell shows the raw key plus optional `displayName`, but no last-message preview or derived title.
  - file: [ui/src/ui/views/sessions.ts:380](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/ui/src/ui/views/sessions.ts#L380)
