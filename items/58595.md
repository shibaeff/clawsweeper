## Summary

Keep open. Current main still carries `esbuild` in the install/build dependency set, still locks `@esbuild/darwin-*` at `0.27.7`, and still documents macOS support without a minimum macOS version. I found no repo evidence that Catalina compatibility was restored or that the docs were updated to declare Catalina unsupported.
