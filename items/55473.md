## Summary

Keep open. Current `main` still treats startup secret-resolution failures as fatal, there is no channel-specific degraded-startup path in the checked-out code, existing tests still assert that unresolved active channel refs fail, and the maintainer discussion on this PR explicitly says the underlying bug is still real even though this branch is not ready to merge.
