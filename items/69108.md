## Summary

Keep open. Current `main` already appears to have the `--session-id` precedence fix when `--agent` is present, but it still does not implement the PR’s `--new-session` flag or its output/docs plumbing, so the PR is only partially superseded.

## Evidence

That covers the bug-fix half of the PR on current `main`.
