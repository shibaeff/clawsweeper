## Summary

Keep open. Current main has real mitigations for the underlying pain: pre-compaction memory flush, startup-context on bare `/new` and `/reset`, and status surfaces that show context percentage. But the requested passive `context=X%` signal is still not part of the system-prompt runtime contract or `Runtime:` line, so there is not strong evidence to close this as implemented or obsolete.
