## Summary

Keep open. Current `main` has partial overlap through the runtime event bus (`api.runtime.events.onAgentEvent`) and the global `thinking` stream, but the plugin hook surface requested by this PR is still not implemented: plugin hooks do not include `thinking_start`/`thinking_end`, and `agent_end` on `main` still lacks `tokenUsage` and `toolCallCount`. The proposed PR therefore is not already implemented on `main`, and the provided review history shows unresolved correctness/scope concerns rather than an obsolete request.
