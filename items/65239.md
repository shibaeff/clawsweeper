## Summary

Keep open. Current `main` still documents and persists plaintext-backed secret storage surfaces, and the security threat model explicitly lists encryption at rest as a recommended future mitigation rather than shipped behavior. This is a plausible core security request, not a ClawHub/plugin-scope item, and it is too recent for stale closure.

## Evidence
