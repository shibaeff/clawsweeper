## Summary

Keep open. Current `main` already covers the directive-tag/code-block bug and the `model.input` optional-chaining fix in `model-scan`, but it does not include the minimal-profile tool guard from #42165 and it still has an unsafe `model.input.join("+")` path in the models list flow for discovered/custom provider models.
