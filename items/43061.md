## Summary

Keep open. Current `main` already covers the code-block whitespace fix and the missing-`model.input` handling, but it still violates the documented `tools.profile: "minimal"` contract by implicitly re-adding `exec`/`process` and filesystem tools when `tools.exec` or `tools.fs` is configured. This PR still contains a useful unmerged fix for that remaining bug.
