## Summary

Keep open. Current `main` does not implement clearer Control UI chat dates: message groups still render only time-of-day footers, and the only divider path in chat is the existing compaction marker, not day separators like “Today” or “Yesterday”.

## Evidence

- **Grouped chat footer still renders time only:** Both streaming and normal message-group rendering format timestamps with `toLocaleTimeString({ hour, minute })` and display them in `.chat-group-timestamp`; there is no date/day-separator logic or full-date hover/title path in this renderer.
  - file: [ui/src/ui/chat/grouped-render.ts:241](https://github.com/openclaw/openclaw/blob/9577de2da7dfb6234cf30e3b879940067804e26e/ui/src/ui/chat/grouped-render.ts#L241)
  - command: `nl -ba ui/src/ui/chat/grouped-render.ts | sed -n '236,326p'`
  - sha: [9577de2da7df](https://github.com/openclaw/openclaw/commit/9577de2da7dfb6234cf30e3b879940067804e26e)
- **Divider generation is only for compaction markers:** `buildChatItems()` emits `kind: "divider"` only when `__openclaw.kind === "compaction"`, hard-coding the label to `"Compaction"`. There is no date-bucketing/day-separator generation in the chat item builder.
  - file: [ui/src/ui/chat/build-chat-items.ts:218](https://github.com/openclaw/openclaw/blob/9577de2da7dfb6234cf30e3b879940067804e26e/ui/src/ui/chat/build-chat-items.ts#L218)
  - command: `nl -ba ui/src/ui/chat/build-chat-items.ts | sed -n '213,230p'`
  - sha: [9577de2da7df](https://github.com/openclaw/openclaw/commit/9577de2da7dfb6234cf30e3b879940067804e26e)
- **Chat view only renders provided divider labels:** The chat view renders divider items generically from `item.label`; it does not derive date separators itself. Combined with the builder above, current `main` has no Control UI path that would show “Today”, “Yesterday”, or explicit message-date separators.
  - file: [ui/src/ui/views/chat.ts:889](https://github.com/openclaw/openclaw/blob/9577de2da7dfb6234cf30e3b879940067804e26e/ui/src/ui/views/chat.ts#L889)
  - command: `nl -ba ui/src/ui/views/chat.ts | sed -n '885,896p'`
