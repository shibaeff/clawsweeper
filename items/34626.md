## Summary

Current `main` already covers the ACP/ACPX megathread items: direct ACP bindings for Discord channels and Telegram topics, native `/acp` command routing on those surfaces, queued ACP/background status updates plus queued-message summaries, resuming existing Codex/ACP sessions via `resumeSessionId`, and Discord attachment ingestion into routed agent context.
