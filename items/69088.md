## Summary

The proposal is explicitly scoped as an external ClawHub plugin, and current main already documents and exposes the plugin SDK and hook surfaces needed for it. OpenClaw's vision and plugin docs say optional capabilities like this should ship outside core unless a narrow missing SDK seam is identified.
