## Summary

Keep open. Current `main` at `68315792677f9628d1370d6869366442f1d3ecad` does not implement the PR's user-specific memory isolation: memory tools still operate on agent-wide memory only, path checks still allow general `memory/**` access, and config/docs do not expose a `memory.isolation` setting.

## Evidence

  - file: [extensions/memory-core/index.ts:42](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/extensions/memory-core/index.ts#L42)
