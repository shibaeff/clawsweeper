## Summary

Current `main` already implements automatic memory recall, but as the bundled `memory-lancedb` memory plugin rather than new core runner/config changes. The plugin registers a `before_prompt_build` hook that embeds the prompt, searches memory, and prepends formatted results; the feature is documented and covered by tests, and it shipped in `v2026.4.22` via commit `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`.
