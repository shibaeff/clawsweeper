## Summary

Keep open. Current `main` does not implement legacy flat-array support for `commands.allowFrom`/`tools.elevated.allowFrom`, and the proposed PR is incomplete because runtime lookup and types still assume provider-keyed records. The PR also bundles unrelated Feishu media changes, so there is still useful work here but not strong evidence to auto-close it.
