## Summary

Current main already implements the runtime pieces this report was missing: heartbeat runs are tagged as heartbeat bootstrap runs, lightweight mode filters bootstrap files down to HEARTBEAT.md, and isolated heartbeat sessions exist to avoid transcript growth. The remaining expectation that lightContext should be read from HEARTBEAT.md is not the supported config surface on current main.
