## Summary

Keep open. Current `main` still scopes explicit in-workspace custom collection names, and the existing QMD compatibility code only handles legacy unscoped->scoped renames. The PR's reverse rename is not reflected on `main`, and the provided review comments identify a plausible upgrade regression that the PR diff does not appear to address.
