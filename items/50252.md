## Summary

Current `main` only partially covers this request: session labels are implemented, and assistant identity can vary by agent, but assistant name/avatar are not stored per individual session. The requested per-session assistant avatar/name customization is not implemented on `main`.

## Evidence

- **Session labels exist in the UI:** The sessions view renders an editable `row.label` input and patches the session when it changes.
  - file: [ui/src/ui/views/sessions.ts:518](https://github.com/openclaw/openclaw/blob/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f/ui/src/ui/views/sessions.ts#L518)

  - file: [src/gateway/sessions-patch.ts:219](https://github.com/openclaw/openclaw/blob/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f/src/gateway/sessions-patch.ts#L219)
