## Summary

Keep open. Current `main` still lacks any cache/TTL around provider usage snapshots: `loadProviderUsageSummary` resolves auths and fetches snapshots on every call, the Anthropic provider still delegates directly to `fetchClaudeUsage`, and that helper still performs a live request to `https://api.anthropic.com/api/oauth/usage`. The same uncached `loadProviderUsageSummary` implementation is present in the latest release tag `v2026.4.22`, so this is not implemented on shipped code.
