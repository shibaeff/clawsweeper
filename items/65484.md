## Summary

Current `main` already captures the gateway origin/thread for embedded turns, routes ACP replies back to that origin, and converts embedded ACP turn failures into final error replies instead of only logging them. The relevant ACP dispatch and Slack-routing code landed after the latest release, so this is fixed on `main` but not yet in a tagged release.

## Evidence
