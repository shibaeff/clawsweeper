## Summary

Current `main` already implements cross-channel messaging for the shared message tool via `tools.message.crossContext.allowAcrossProviders`, documents the exact Telegram-to-Discord case, and preserves legacy compatibility for `tools.message.allowCrossContextSend`. The issue request is satisfied on current `main`.
