## Summary

Current OpenClaw already ships a dedicated Render deployment path: the repo includes `render.yaml`, a matching Render install guide, and gateway/container behavior that matches Render's port and health-check model. The latest release `v2026.4.22` also contains that implementation, so the issue is obsolete against current shipped code.
