## Summary

Keep open. Current `main` has adjacent memory work (`memory-core` dreaming, `memory-wiki`, plugin-based memory slots), but it does not implement the RFC's core asks: linked-memory graph storage, graph/PPR retrieval, or link-aware consolidation/forgetting. The report is also recent and detailed enough to remain actionable.
