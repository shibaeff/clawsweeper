## Summary

Current `main` still implements the opposite behavior: orphaned restored/resumed subagent runs are marked as errors and pruned without any transcript-validity check, and shipped tests/documentation codify that behavior. The PR also still has substantive review concerns in the provided discussion, so conservative maintainer cleanup should keep it open.
