## Summary

Current `main` does not contain this PR’s formatter changes. The remaining edge cases in the PR are still plausible on current code because TUI sanitizes raw assistant text directly, and `isCopySensitiveToken` still classifies wrapped/dotted tokens using the old narrower rules.
