## Summary

Keep open. Current `main` already has an equivalent user-facing Slack toggle (`channels.slack.thread.requireExplicitMention`) with docs and tests, but the PR also fixes a real multi-account merge gap that is still present on `main`: Slack account config merging remains shallow for `thread`, so an account-level `thread` override can wipe out a global thread mention setting.
