## Summary

Current `main` still reinjects AGENTS.md-derived post-compaction context by default, including legacy heading fallback, while the normal system prompt already carries AGENTS.md project context and is expected to stay stable across normal turns. I found no evidence that #45488 is implemented on `main`; the report remains actionable.
