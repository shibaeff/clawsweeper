## Summary

Keep open. Current `main` has a real macOS companion app and release machinery for macOS, but the broader request is not implemented: OpenClaw still documents itself as terminal-first, the install path is still script/CLI based, and Windows desktop installers/companion app are explicitly still planned rather than shipped.

## Evidence
