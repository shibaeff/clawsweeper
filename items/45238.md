## Summary

Current main already has a bundled-plugin runtime-dependency repair path for missing browser `playwright-core`, documents the operator workflow, and ships release checks that specifically verify the browser plugin can recover that dependency in packed installs. The original report's suspected pw-ai bundle-resolution bug no longer matches the current supported runtime-deps flow on main.
