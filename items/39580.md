## Summary

Keep open. Current `main` still has the mismatch the PR fixes: node-service installs set `OPENCLAW_SYSTEMD_UNIT` to `openclaw-node`, but Linux install/uninstall still enable/disable the gateway unit name instead of the overridden unit. The PR remains a valid minimal fix, and the existing workaround comment aligns with the current code path.

## Evidence
