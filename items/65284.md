## Summary

Keep open. Current `main` still contains the provider-collision surfaces this PR targets, but the PR's root-cause change also conflicts with the documented/tested `models.mode: "merge"` contract that preserves existing `models.json` providers. This is not safely auto-closable; it needs maintainer judgment or a narrower fix.
