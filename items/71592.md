## Summary

Current `main` still advertises `/status` and `/compact` in TUI slash-command discovery, but the TUI command handler has no local cases for either command and falls through to raw chat send. In `--local` mode that raw send goes straight into the embedded run path, so the reported behavior remains plausible and supported by the codebase.
