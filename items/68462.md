## Summary

Keep open. Current `main` still auto-restores `openclaw.json.bak` in `maybeRecoverSuspiciousConfigRead*` without checking the backup for redacted-placeholder pollution, so this PR is not implemented on `main`. A related safeguard added on `main` only protects `.last-good` promotion/recovery, not the `.bak` restore path this PR changes.
