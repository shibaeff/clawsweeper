## Summary

Keep open. Current `main` still implements `pnpm check:docs` through a Unix-style `git ... | xargs oxfmt` pipeline, so the PowerShell/Windows compatibility request has not been implemented on `aa21d4ea7e9aab034acbf0b13f8c3575057c5c54`.
