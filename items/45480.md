## Summary

Current `main` already removes the reported TDZ path. Anthropic model normalization now goes through `src/agents/model-ref-shared.ts`, where aliases are handled by a local `switch` instead of a later-initialized `ANTHROPIC_MODEL_ALIASES` constant, and the same implementation is present in the `v2026.4.22` release snapshot.
