## Summary

Current `main` does not implement this PR’s behavior. `openclaw logs` still defaults to UTC unless `--local-time` is passed or `process.env.TZ` is set to a valid zone, and both docs and tests still encode that old contract. This PR remains a live change rather than cleanup for already-shipped behavior.
