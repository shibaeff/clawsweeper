## Summary

Current `main` does not implement delivery-context-based session resolution for `openclaw agent`. The CLI still derives the session key from `--to`/`--session-id` only, while `--channel` and reply/account flags are treated as delivery-only. That matches the reported bug rather than proving it fixed.

## Evidence

- **CLI flags still separate routing from delivery:** `src/cli/program/register.agent.ts:27-42` defines `--to` as the value that derives the session key, while `--channel`, `--reply-to`, `--reply-channel`, and `--reply-account` are described as delivery options, not session selectors.
  - file: [src/cli/program/register.agent.ts:27](https://github.com/openclaw/openclaw/blob/aa21d4ea7e9aab034acbf0b13f8c3575057c5c54/src/cli/program/register.agent.ts#L27)
  - sha: [aa21d4ea7e9a](https://github.com/openclaw/openclaw/commit/aa21d4ea7e9aab034acbf0b13f8c3575057c5c54)
- **Gateway request resolves session key without channel/account context:** `src/commands/agent-via-gateway.ts:113-143` calls `resolveSessionKeyForRequest({ cfg, agentId, to: opts.to, sessionId: opts.sessionId })`, then sends `channel` and `replyAccountId` separately afterward. The resolver never receives channel/account delivery context.
  - file: [src/commands/agent-via-gateway.ts:113](https://github.com/openclaw/openclaw/blob/aa21d4ea7e9aab034acbf0b13f8c3575057c5c54/src/commands/agent-via-gateway.ts#L113)
  - sha: [aa21d4ea7e9a](https://github.com/openclaw/openclaw/commit/aa21d4ea7e9aab034acbf0b13f8c3575057c5c54)
- **Resolver API still has no delivery-context lookup path:** `src/agents/command/session.ts:130-155` accepts only `to`, `sessionId`, `sessionKey`, and `agentId`; it derives `ctx` from `opts.to` alone. There is no `channel`/`accountId`/`threadId` parameter or search by stored `deliveryContext`.
  - file: [src/agents/command/session.ts:130](https://github.com/openclaw/openclaw/blob/aa21d4ea7e9aab034acbf0b13f8c3575057c5c54/src/agents/command/session.ts#L130)
  - sha: [aa21d4ea7e9a](https://github.com/openclaw/openclaw/commit/aa21d4ea7e9aab034acbf0b13f8c3575057c5c54)

  - file: [src/commands/agent/session.test.ts:62](https://github.com/openclaw/openclaw/blob/aa21d4ea7e9aab034acbf0b13f8c3575057c5c54/src/commands/agent/session.test.ts#L62)
