## Summary

Keep open. Current `main` does not already implement Feishu chat-history listing, and the PR still carries an unresolved correctness issue called out in review: it falls back from `threadId` to a helper that hardcodes `container_id_type: "chat"`, which is not the same Feishu identifier type as a thread/topic lookup.
