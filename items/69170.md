## Summary

Current `main` enforces `streamTo` as ACP-only at runtime, but it still does not document that requirement in the `sessions_spawn` tool schema. This PR's schema-hint change is therefore not implemented on `main`, so it should stay open unless a maintainer closes it as superseded by a different PR with the same missing change.
