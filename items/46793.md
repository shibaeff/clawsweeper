## Summary

Keep open. Current `main` at `68315792677f9628d1370d6869366442f1d3ecad` still leaves direct `/skill` inline tool dispatch outside the normal tool-policy pipeline. The inline path in `src/auto-reply/reply/get-reply-inline-actions.ts` builds tools and applies only `applyOwnerOnlyToolPolicy` before executing the matched tool, while the normal agent tool path in `src/agents/pi-tools.ts` runs `applyToolPolicyPipeline` with profile/provider/agent/group/sandbox/subagent policy inputs. The PR's regression test file is also absent on `main`, so there is no evidence that this fix landed elsewhere.
