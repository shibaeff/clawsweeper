## Summary

Keep open. Current `main` still builds the managed gateway PATH from a hardcoded minimal directory list and does not include Nix Home Manager paths such as `~/.nix-profile/bin` or anything derived from `NIX_PROFILES`. `env.shellEnv` also does not repair this for daemon installs because it only imports expected auth/channel env keys, not `PATH`.

## Evidence

- **Gateway install PATH is synthesized from fixed helpers:** `buildCommonServiceEnvironment()` writes `PATH` from `sharedEnv.minimalPath`, and `resolveSharedServiceEnvironmentFields()` sets that from `buildMinimalServicePath(...)` for non-Windows installs.
