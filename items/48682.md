## Summary

Current `main` still has the linked bug, but this PR is not ready to close out the item. GitHub Copilot implicit discovery still ignores `auth.order` on `main`, and the PR’s current approach moves provider-specific logic into core while introducing an unresolved plugin-disable regression called out in review.
