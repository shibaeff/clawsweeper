## Summary

Keep open. Current `main` has partial bundled-MCP cleanup for embedded/local and isolated cron runs, but the heartbeat and queued follow-up paths this PR targets are still not wired for one-shot runtime cleanup, so the PR is not implemented on `main` yet.
