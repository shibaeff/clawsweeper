## Summary

Current `main` already contains a browser-microphone fix for web surfaces, and the changelog shows it shipped after the reported `2026.4.11` version in `v2026.4.18`. The webchat voice button is still wired to browser STT, while gateway HTTP headers now explicitly allow same-origin microphone access and have regression coverage.
