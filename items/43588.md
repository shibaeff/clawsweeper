## Summary

Keep open. Current `main` at `9577de2da7dfb6234cf30e3b879940067804e26e` still implements Slack `allowBots` as boolean-only across config types, schema, runtime gating, and generated config metadata, so the PR’s `"mentions"` mode is not already implemented or obsolete. The request is also coherent: Discord already supports the same `allowBots: "mentions"` pattern, so this remains a real Slack parity gap rather than a stale cleanup candidate.
