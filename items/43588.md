## Summary

Keep open. Current `main` does not implement Slack `allowBots: "mentions"` parity: Slack config types and schemas are still boolean-only, Slack inbound handling still treats `allowBots` as a simple boolean, and the Slack tests on `main` do not cover mention-only bot handling. The PR’s original file paths are stale relative to today’s `extensions/slack/*` layout, but the requested behavior is not already on `main`.
