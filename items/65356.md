## Summary

Keep open. Current `main` still stores sandbox backends in a module-local `Map`, while sandbox backend registration is a plugin-facing SDK surface and plugin source loads go through `jiti`. That makes the reported split-registry failure plausible on current code, and there is no evidence in `main` that the PR was superseded or already implemented.
