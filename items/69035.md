## Summary

Current `main` already handles the reported `@homebridge/ciao` interface assertion on the Bonjour/mDNS startup path by classifying it as a non-fatal interface assertion, logging a warning, and suppressing the unhandled rejection instead of crashing the gateway. The exact error text from the report is covered by focused tests, and the fix is present on `main` but not in the latest tagged release.

## Evidence
