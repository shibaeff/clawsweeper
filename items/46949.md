## Summary

Keep open. Current `main` does not contain the PR's core behavior: startup reconciliation still leaves oneshot handles cached, concurrent-session enforcement still hard-fails instead of reclaiming dormant runtimes, and the existing tests still encode the old rejection behavior. The PR discussion also shows useful follow-up work rather than an obsolete report.
