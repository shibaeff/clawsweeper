## Summary

Kokoro TTS is a reasonable request, but current OpenClaw architecture puts optional provider integrations in plugins rather than adding more bundled core providers. The repo already documents and supports external speech/provider plugins, and current main also offers a Kokoro-based workaround through the bundled OpenRouter speech provider.
