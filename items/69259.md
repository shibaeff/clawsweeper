## Summary

Keep open. Current `main` does not already include the requested GitHub Copilot default-model expansion, and the present tests explicitly lock in at least one of the requested IDs as absent. OpenClaw does have forward-compatible fallback logic for manually configured Copilot model IDs, but that is not the same as adding those IDs to the default list this PR targets. The related duplicate PRs mentioned in the timeline are still open in the provided GitHub context, so there is no landed replacement that would justify closing this PR.
