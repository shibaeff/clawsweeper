## Summary

Keep open. Current `main` still enforces a 2 MiB limit for local agent avatars and rejects oversized files, which can surface as a 404 on the avatar route, while the user-facing avatar docs still describe accepted avatar value types without mentioning the size cap. That leaves the reported docs gap unresolved.
