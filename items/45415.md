## Summary

Keep open. Current `main` has partial bootstrap-size diagnostics and truncation warnings, but it does not implement the requested Control UI size indicator or save-time MEMORY.md limit enforcement. The remaining behavior is still missing on `main`, so there is not strong evidence to close this as already implemented.

## Evidence

- **Save path does not enforce a MEMORY.md/bootstrap size limit:** `agents.files.set` validates params, writes the file, and returns success metadata. There is no check against `bootstrapMaxChars`, no warning threshold handling, and no refusal-to-save path.
