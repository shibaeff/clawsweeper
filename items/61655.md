## Summary

The request proposes a reusable authoring/composition layer for workspace instructions (`AGENTS.md`, `SOUL.md`, etc.), but current main keeps workspace files intentionally flat and the project vision pushes optional configuration/skill packaging to plugins/ClawHub rather than new core orchestration machinery.

## Evidence
