## Summary

Current main already includes the Control UI history-rendering fix for uploaded user images, and it shipped in v2026.4.23. Gateway persists uploaded image paths into transcript history, and the Control UI now re-renders those history-backed images instead of dropping them after the post-send history reload.

## Evidence
