## Summary

Keep open. Current `main` still allows `/phone arm` and `/phone disarm` from external channels when `gatewayClientScopes` is absent, which is the behavior this PR is trying to remove. The existing tests on `main` explicitly preserve that external-without-admin path, so this is not already implemented.

## Evidence
