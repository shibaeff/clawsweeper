## Summary

Current `main` already fixes the Control UI image-attachment regression behind this report: the dashboard chat still captures pasted/dropped/selected images, sends attachment-only turns through `chat.send`, and the reply pipeline now treats inline images as media instead of dropping image-only turns as empty. The changelog records that fix in the 2026.4.20 release.
