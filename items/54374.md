## Summary

Current `main` does not contain this fix. Gateway chat deltas still broadcast the full accumulated assistant snapshot after later segments, the web UI still commits pre-tool stream text above the tool card and then renders the next delta as the live stream below it, and current tests explicitly lock in the full-snapshot behavior for non-prefix later segments. This PR should stay open for maintainer judgment rather than be auto-closed.
