## Summary

Current `main` has warm-load mitigations for bootstrap files and plugin registries, but the reported hot path is not clearly eliminated: workspace bootstrap loading still performs synchronous `readFileSync` calls, and plugin-registry caching is scoped by `workspaceDir`/config, so a multi-agent setup with distinct workspaces can still miss the cache and reinitialize plugins separately. I do not have high-confidence evidence that issue #65328 is already fixed on `main`.
