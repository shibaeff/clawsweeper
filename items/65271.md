## Summary

Keep open. Current `main` still hardcodes PowerShell on Windows and the shipped docs describe that behavior, so this PR is not implemented on `main`. The PR also has unresolved review concerns around invalid/misdetected shell overrides, so there is not strong evidence for a conservative close.
