## Summary

Keep open. Current `main` still ships the bundled `bonjour` plugin on `@homebridge/ciao` 1.3.6 without the proposed patch, the upstream Windows code path still omits `windowsHide`, and the linked bug report matches that shipped behavior. This PR remains a concrete, actionable fix rather than something already implemented or obsolete.
