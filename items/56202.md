## Summary

Keep open. Current `main` has partial EnvironmentFile support for systemd gateway installs, but it does not fully subsume this PR: it writes a state-dir env file only for dotenv-backed values, keeps some values inline, and `uninstallSystemdService` still removes only the unit file.
