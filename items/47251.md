## Summary

Keep open. Current `main` still intentionally ships `node-llama-cpp` as an optional peer for npm releases, and the release checks/tests enforce that policy. The docs mention local embeddings and pnpm native-build approval, but they do not document an npm-global manual `node-llama-cpp` install path, so the reported packaging gap remains plausible rather than implemented or obsolete.
