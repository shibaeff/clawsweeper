## Summary

Keep open. Current main still injects a blanket same-group message-tool prohibition into every group-chat auto-reply session, while Telegram still has explicit same-chat topic auto-threading support for tool sends. That matches the reported contradiction, so there is not strong evidence to close this issue as fixed or obsolete.
