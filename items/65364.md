## Summary

Current `main` does not implement the proposed `registerProviderRuntimeAuthOverride` seam or its supporting runtime/auth plumbing, and the existing plugin hooks only cover provider/model selection, not runtime credential injection. The PR discussion shows the author addressed the main review findings rather than the change becoming obsolete, so conservative cleanup should keep this PR open.
