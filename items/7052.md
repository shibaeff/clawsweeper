## Summary

Current main already covers the request: OpenClaw now does automatic threshold/overflow compaction, runs a silent pre-threshold memory flush before compaction, keeps compaction checkpoints, and repairs orphaned tool_result/tool_use state to avoid the reported API rejection path.
