## Summary

Keep open. Current `main` still does not support populating content during `feishu_doc` `create`: the `create` schema has no `content` field, and the runtime path only calls `createDoc(...)` without any follow-up write. The PR appears to address a real bug. One review comment is obsolete on current `main`, though: `feishu_doc` is already wired into Feishu plugin startup via `extensions/feishu/index.ts`.
