## Summary

Keep open. Current `main` and the latest shipped release still include the legacy `imsg` iMessage integration as a bundled channel with docs, config surface, and tests, but the docs explicitly still say it is legacy and "may be removed in a future release." That means the user's request for continued retention/support is not actually resolved on `main`; it remains an open product/maintenance decision rather than something already implemented or obsolete.
