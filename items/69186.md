## Summary

Current `main` does not implement a Control UI completion/success notification for agent turns. The chat settings surface only exposes thinking/tool-call/focus toggles, the persisted UI settings schema has no notification/sound field, and the Control UI chat event pipeline clears terminal runs without any browser audio/notification hook. This remains an unimplemented, in-scope Control UI feature request.
