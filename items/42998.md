## Summary

Keep open. Current `main` does not include the raw-stdout/tool-result recovery proposed by PR #42998, but the PR also has unresolved correctness concerns from review. The report as written no longer matches `main` exactly because empty payloads are not treated as interim acknowledgements anymore, so there is not high-confidence evidence for auto-closing this PR.
