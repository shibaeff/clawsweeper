## Summary

Keep open. Current `main` already propagates `enabledByDefault` through plugin skill activation, but `open-prose` itself is still intentionally opt-in on `main`: its manifest does not set `enabledByDefault: true`, and both the plugin README and public prose docs tell users to enable the plugin manually because bundled plugins are disabled by default. That means this PR's core product change is not implemented on `main`, so it is not a safe cleanup close.
