## Summary

Current `main` and the released `v2026.4.23` build already route Docker `sandbox.docker.binds` through the default sandbox file bridge, with tests covering absolute mounted paths and bind precedence, so the 2026.4.15 regression report is obsolete.

## Evidence

- **Sandbox context hands Docker bind config to the file bridge:** The sandbox context stores `resolvedCfg.docker` and then instantiates the default file bridge when the backend does not override it, which is the Docker-backend path relevant to this report.
  - file: [src/agents/sandbox/context.ts:228](https://github.com/openclaw/openclaw/blob/9577de2da7dfb6234cf30e3b879940067804e26e/src/agents/sandbox/context.ts#L228)
