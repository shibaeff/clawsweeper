## Summary

Current `main` now documents and wires a first-class pre-turn plugin hook for deterministic model/provider overrides (`before_model_resolve`), which covers the core request to document and support intent-based model routing as an OpenClaw pattern. The runtime applies the hook before model resolution, the docs now point plugin authors at it for model switching, and the issue-linked external router fits the intended plugin/ClawHub extension path.

## Evidence
