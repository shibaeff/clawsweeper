## Summary

Keep open. Current `main` still intentionally dispatches agent followups after denied exec approvals, still drops queued messages when the cap is hit without any stop-command priority carveout, and the issue’s referenced fix appears to exist only as the still-open PR #69387 rather than code on `main`.
