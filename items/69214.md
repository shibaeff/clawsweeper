## Summary

Close as implemented on main. Current pairing code merges approved scope upgrades into the paired device record, rotates the relevant device token, deletes the pending request, and keeps repeated reconnect attempts reconciled to a single pending request. The related pairing/doctor/UI fixes are documented in the v2026.4.23 changelog, which matches the reported symptom area.
