## Summary

Keep open. Current main does not show a clear fix for this report, and the reported failure mode is still plausible: the `gog` skill is still a normal model-routed skill rather than a deterministic command-dispatch tool, so there is no strong evidence that browser `/skill gog ...` now bypasses the path that could surface `sessions.resolve` failures.
