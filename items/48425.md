## Summary

Keep open. Current `main` still writes `skillsSnapshot` into `sessions.json`, so the reported bloat is not implemented/fixed on `main`. This PR also has an unresolved correctness risk: its replacer strips any nested property named `skillsSnapshot`, which can silently drop legitimate persisted ACP `backendExtras` data.
