## Summary

Keep open: current `main` still hard-couples `mode="session"` to `thread=true`, and thread-bound spawning still depends on channel `subagent_spawning` hooks. Local embedded mode also disables `sessions_spawn` entirely, so the requested decoupling for local persistent subagent sessions is not implemented. The issue body is only partially current because `main` now has Feishu/Matrix subagent hook surfaces, but that does not resolve the core feature request.
