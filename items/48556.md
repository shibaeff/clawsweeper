## Summary

Keep open. Current `main` still routes proxy-wrapped requests straight into `undici.fetch` without normalizing `FormData`, while the affected core call sites still build plain global `FormData` and pass it through the proxy fetch path. The existing Discord-only `UndiciFormData` workaround supports the PR’s diagnosis but does not cover the generic core wrapper.
