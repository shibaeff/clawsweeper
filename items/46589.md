## Summary

Keep open. Current `main` already has the cron edit and run-history UI this PR originally bundled, and the old root-Vitest registration/jsdom rationale is partly obsolete because UI tests now run under a dedicated jsdom project. But the PR's remaining substantive work is not on `main`: the cron pipeline-chaining fields and dispatch path are absent, and the i18n fixes in the PR are also still missing. That is not strong enough evidence for any close reason.
