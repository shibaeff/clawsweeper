## Summary

Keep open. Current `main` still leaves the Chat view only partially localized, so this request is not implemented. The PR’s direction is useful, but the submitted approach is not ready for cleanup closure because it relies on a `fallback` parameter that the current `t()` implementation does not honor, and it hand-edits generated non-English locale bundles.
