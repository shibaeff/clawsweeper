## Summary

Keep open. Current `main` already covers the `config.patch` hot-reload half of this PR, but it does not implement the MiniMax XML-to-tool-call promotion half. On `main`, MiniMax `<minimax:tool_call>` XML is still stripped from assistant text rather than converted into structured tool blocks, so this PR is not obsolete.
