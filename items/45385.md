## Summary

Keep open. Current `main` still rewrites unmatched Anthropic `invalid_request_error` payloads as `LLM request rejected: <raw message>`, so the user-facing leak described in #45385 is still present. The repo does contain stronger Anthropic turn repair for dangling trailing tool calls, but that is only a partial mitigation and does not close the formatting gap in the issue title.
