## Summary

Keep open. Current `main` still treats `memory.qmd.paths[*].path` as the collection root verbatim, so a direct file target becomes `path=<file>, pattern=**/*.md` instead of a parent-directory collection with an exact filename. The PR addresses a real gap and adds the missing regression coverage, but the review thread also surfaced a remaining edge case: literal filenames with glob metacharacters would need escaping before being used as a QMD pattern.
