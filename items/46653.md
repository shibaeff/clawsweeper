## Summary

Keep open. Current `main` still does not implement `openclaw tunnel`; docs still tell users to run a manual `ssh -N -L ...` tunnel; the existing SSH support is transient inside `gateway probe/status`; and the PR discussion already surfaced unresolved correctness and safety bugs. The branch is also stale against the current CLI registration shape, so this is not a conservative cleanup close.
