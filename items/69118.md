## Summary

Keep open. Current `main` still hashes the first-turn-only group intro into `extraSystemPromptHash`, so a group channel turn-1 -> turn-2 transition can still change the reuse key and invalidate Claude CLI sessions with `reason=system-prompt`. `main` also still treats missing stored prompt hashes as mismatches, which preserves the legacy-binding failure mode described in the issue.
