## Summary

Keep open. Current `main` still does not bridge tool execution into internal hooks, so this PR is not implemented or obsolete. The repo still limits internal hooks to command/session/agent/gateway/message events, while tool observation remains plugin-hook-only. The discussion also shows the author addressed the earlier syntax/double-fire/sessionKey review concerns in the PR branch, so this is still a live, reviewable change rather than an incoherent or stale cleanup candidate.
