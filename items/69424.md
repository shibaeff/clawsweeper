## Summary

Keep open. The reported regression still appears present on current `main` and in the latest tagged release: the npm package whitelist still ships only `docs/reference/templates/`, while agent/runtime code still resolves `${packageRoot}/docs` and tells models to consult local docs first. `openclaw docs` now searches the live docs index, but that is a separate CLI path and does not fix local-doc consumers like `openclaw-docs sync` or `docs-search` workflows that read the bundled `docs/` tree.
