## Summary

Current `main` already restructures gateway startup to avoid the old silent pre-log load path: the CLI now treats server-module import as a distinct startup phase, the public gateway entrypoint lazy-loads the heavy server implementation, and startup emits incremental log lines before and after HTTP bind/sidecar startup.
