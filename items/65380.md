## Summary

Keep open. Current `main` still sends MiniMax Anthropic-compatible traffic through the shared hybrid replay policy, which enables strict tool-call ID sanitization, and the current tests on `main` explicitly lock that behavior in. I did not find evidence of a landed or shipped fix in the inspected code.
