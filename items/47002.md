## Summary

Keep open. Current `main` appears to have fixed the underlying host-local outbound media failure through fs-policy-based media-root widening, but it still does not add `mediaLocalRoots` to the Telegram or `agents.defaults` config schema/type surface described in the issue, so this is not a strong cleanup close.
