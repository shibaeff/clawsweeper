## Summary

Keep open. Current `main` still has the reported Slack prompt mismatch, so this is not `implemented_on_main` or `cannot_reproduce`. The PR is directionally useful and adds the missing regression test, but the discussion shows one remaining correctness gap: the disabled-path Slack hint still tells users to enable `inlineButtons` even though Slack's documented/runtime capability is `interactiveReplies`.
