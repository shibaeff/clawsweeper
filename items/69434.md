## Summary

Keep open. Current `main` still treats `agents.*.heartbeat.model` as a single string override, not a `{ primary, fallbacks }` selection, and the heartbeat path only forwards/resolves one model ref. The linked open PR #69495 in the issue timeline also indicates this is still active work rather than a stale or already-landed change.
