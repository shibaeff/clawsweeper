## Summary

Keep open. Current `main` does not implement per-binding `groupScope`, still documents groups/channels as isolated from the main session, and still lacks the IRC active-client send reuse included in this PR. Closing as implemented or obsolete would be incorrect; if anything, the PR still needs follow-up for the outbound and Telegram topic-path concerns raised in review.
