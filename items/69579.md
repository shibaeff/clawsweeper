## Summary

Current `main` already preserves Anthropic signed thinking blocks across follow-up replay for Anthropic-compatible turns, including custom-proxy/native Anthropic paths, and has regression coverage for immutable signed-thinking replay safety. I could confirm the implementation on `0bef73d151bd0a8d547b28bf16c3aa2e21078e0f`, but I could not tie it to a shipped release from this shallow checkout, so `fixedRelease` is left unset.
