## Summary

Keep open. Current `main` still routes Control UI/WebChat dictation and read-aloud through browser speech APIs, and there is still no gateway STT/transcription RPC for the web client to call, so the requested behavior is not implemented.

## Evidence

- **WebChat speech layer is still browser-native:** `ui/src/ui/chat/speech.ts` explicitly says STT uses `SpeechRecognition` and TTS uses `SpeechSynthesis`, then implements `getSpeechRecognitionCtor()` from `window.SpeechRecognition` / `window.webkitSpeechRecognition` and `speakText()` via `speechSynthesis.speak(...)`.
  - file: [ui/src/ui/chat/speech.ts:2](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/ui/src/ui/chat/speech.ts#L2)
  - command: `nl -ba ui/src/ui/chat/speech.ts | sed -n '1,260p'`
  - sha: [028b6c9b13c9](https://github.com/openclaw/openclaw/commit/028b6c9b13c960b6f59a7643aff4f6de04e07f71)
- **Mic button still calls browser STT helper:** The chat composer imports `startStt`/`stopStt` from `../chat/speech.ts`, and the mic button starts/stops that helper directly instead of calling a gateway transcription method.
  - file: [ui/src/ui/views/chat.ts:34](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/ui/src/ui/views/chat.ts#L34)
  - command: `nl -ba ui/src/ui/views/chat.ts | sed -n '1248,1310p'`
  - sha: [028b6c9b13c9](https://github.com/openclaw/openclaw/commit/028b6c9b13c960b6f59a7643aff4f6de04e07f71)
- **Read-aloud button still calls browser TTS helper:** The assistant message `Read aloud` button invokes `speakText(text, ...)`, which is the browser `speechSynthesis` path above, not gateway `tts.convert`.
  - file: [ui/src/ui/chat/grouped-render.ts:589](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/ui/src/ui/chat/grouped-render.ts#L589)
  - command: `nl -ba ui/src/ui/chat/grouped-render.ts | sed -n '560,640p'`

Audio transcription exists as backend runtime plumbing (`src/media-understanding/transcribe-audio.ts` re-export), not a web-facing gateway method.
