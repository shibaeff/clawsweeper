## Summary

Current `main` already addresses this Discord auto-thread title failure: the generated-title helper now uses a 512-token budget specifically to avoid reasoning-model starvation, the rename flow still routes through that helper, and the same implementation is present in stable release `v2026.4.23`.

## Evidence
