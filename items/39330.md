## Summary

Keep open. Current `main` still intentionally appends the unscheduled reminder note to user-visible replies, and the current tests assert that behavior. The PR is therefore not implemented or obsolete, and the review thread still has an unresolved correctness concern about stripping legitimate payload text. This needs maintainer judgment, not conservative stale cleanup.
