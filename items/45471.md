## Summary

Keep open. The report narrowed to Control UI auth/token drift and gateway-instance confusion, but current `main` still shows an unresolved recovery UX gap rather than a clear implemented fix. `openclaw dashboard` can emit a tokenized `#token=` URL and SecretRef guidance, while `openclaw gateway status` still prints only a plain dashboard URL, matching the linked follow-up PR description rather than a shipped fix.
