## Summary

Keep open. Current `main` still limits Control UI uploads to image MIME types, drops non-image files in the gateway attachment pipeline, and stores only image-style `dataUrl` attachments in the chat UI. This is not implemented on `main`, not stale, and it is core/web-frontend scope rather than a ClawHub-only plugin request.
