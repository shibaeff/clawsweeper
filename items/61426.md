## Summary

Current `main` still defaults isolated `agentTurn` cron jobs without an explicit `delivery` block to announce delivery, and the isolated cron delivery path still queues awareness text into the main session. That matches the core behavior described in the issue closely enough that there is not strong evidence to close it as already fixed or obsolete.

## Evidence
