## Summary

Keep open. Current `main` does not already implement this frontmatter validation, but the claimed tar-flag injection path is not supported by the current code flow, and the submitted patch appears to introduce a behavior regression by making explicit `archive` values case-sensitive. For conservative cleanup, that is not strong enough evidence to close the PR.
