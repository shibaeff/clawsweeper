## Summary

Keep open. Current `main` and the latest release still fall back to the raw Feishu agent id for card headers and always send that header on both streaming and static card paths, so the reported UX bug remains live. The PR is coherent, narrowly scoped, and its second commit addresses the trailing-space edge case raised in discussion, but it still needs normal maintainer review and validation rather than cleanup closure.
