## Summary

Current `main` already supports Feishu group auto-response without @mentions. The implementation uses normal inbound Feishu message events plus `groupPolicy: "open"` / `requireMention: false`, not a polling loop, but it satisfies the requested behavior. This appears to be on `main` only and not yet in released `v2026.4.22`.

## Evidence

  - file: [extensions/feishu/src/monitor.account.ts:327](https://github.com/openclaw/openclaw/blob/4f4d2ef1dffd27fc7486850185af16429061606d/extensions/feishu/src/monitor.account.ts#L327)
