## Summary

Keep open. Current `main` has broad `openclaw doctor` migrations and health checks, but it does not implement a `--check-config` flag or an equivalent deep config-validation pass that exercises config-dependent runtime paths like TTS/provider initialization or context-pruning/model-normalization startup failures.
