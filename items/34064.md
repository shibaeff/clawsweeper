## Summary

Current `main` does not implement Feishu message-history retrieval for agents. The bundled `feishu_chat` tool still only exposes `members`, `info`, and `member_info`, and the Feishu channel action registry also has no `history` action to fetch prior chat messages.

## Evidence

- **`feishu_chat` schema excludes `history`:** The allowed action enum is `members`, `info`, and `member_info` only, and the schema description matches that list.
  - file: [extensions/feishu/src/chat-schema.ts:3](https://github.com/openclaw/openclaw/blob/9153598e659daf5a52c479731f5b4ab8a5bf16b7/extensions/feishu/src/chat-schema.ts#L3)
  - command: `nl -ba extensions/feishu/src/chat-schema.ts | sed -n '1,40p'`
  - sha: [9153598e659d](https://github.com/openclaw/openclaw/commit/9153598e659daf5a52c479731f5b4ab8a5bf16b7)
- **`feishu_chat` implementation has no history branch:** The registered tool description says `Actions: members, info, member_info`, and the switch handles only those cases before falling through to `Unknown action`.
  - file: [extensions/feishu/src/chat.ts:147](https://github.com/openclaw/openclaw/blob/9153598e659daf5a52c479731f5b4ab8a5bf16b7/extensions/feishu/src/chat.ts#L147)
