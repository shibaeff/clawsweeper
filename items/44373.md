## Summary

Keep open. Current `main` already covers part of the PR intent by rendering Overview stat panels as native clickable buttons wired to tab navigation, but it does not fully subsume the PR: non-interactive `.card` and `.account-card` elements still advertise hover affordances globally, and there is no current test coverage for Overview card keyboard activation in the checked-out tree.
