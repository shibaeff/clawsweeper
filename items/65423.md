## Summary

Keep open. Current `main` still feeds unlocked runs a deterministic `profileOrder` with no subagent-specific shuffle, so the proposed thundering-herd mitigation is not implemented on `main`. The PR also has an unaddressed review finding that its new test redefines `shuffleArray` locally instead of exercising the production code path.
