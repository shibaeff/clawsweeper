## Summary

Keep open. Current `main` still appears to miss the reported daily/scheduled session-memory flush path, and the PR's latest approach is not a conservative close candidate because it repurposes command hooks for automatic rollovers and conflicts with current command-authorization and hook semantics.

There is no current-main branch for stale scheduled rollovers where `resetTriggered` is false.
