## Summary

Keep open. Current `main` does not include the `iMessageLite` -> `iMessage` normalization from PR #61561, so the reported BlueBubbles satellite-message gap is not implemented on `main`. The PR is still a plausible, narrowly scoped fix; conservative cleanup does not support closing it.
