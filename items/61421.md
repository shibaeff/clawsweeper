## Summary

Current main and the shipped v2026.4.23 release now expose the Dreaming schema, `/dreaming` alias, `openclaw memory promote` CLI surface, and the runtime fixes for managed cron reconciliation and narrative `idempotencyKey`, so the docs/package mismatch reported for 2026.4.2 is no longer present.
