## Summary

Keep open. Current `main` has added session-scoped effective-tool visibility (`/tools`, `tools.effective`, Control UI) and some reason-specific sandbox/exec guidance, but it does not fully implement the issue's request: there is no strict inheritance mode, and the runtime-effective inventory still does not expose declared tools, removed tools, or a unified clipping-source trace.
