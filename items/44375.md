## Summary

Keep open. Current `main` still falls back to `agents.list[].default` or else the first `agents.list` entry, so adding a single ACP agent without an explicit `main` entry still changes the default agent for unmatched traffic. I did not find a current warning or validation in the `config set` path that would surface this foot-gun before writing config.
