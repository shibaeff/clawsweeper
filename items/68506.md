## Summary

Keep open. Current `main` does not include the proposed degraded-state recovery guidance for `status --all`, the new coverage in the PR is not present on `main`, and the existing review discussion only surfaces minor robustness issues rather than a reason to close the PR.

It does not classify those states into targeted `Cause:` / `Fix:` guidance as described in PR #68506.
