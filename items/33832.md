## Summary

Keep open. Current `main` does not show the requested generic Gateway model API timing/size logging. The only nearby observability on the model-call path is cache tracing plus an Anthropic-only payload/usage logger, and the Gateway docs expose `--raw-stream` and startup tracing but no `--log-timing` or equivalent request/response latency+size log surface.
