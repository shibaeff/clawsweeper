## Summary

Keep open. Current `main` still lacks the Feishu doctor compatibility migration this PR proposes, while the schema and docs already expect `channels.feishu.accounts.*.name` rather than legacy `botName`. Code inspection shows no equivalent implementation landed on `main`, so this remains a live fix rather than cleanup-close material.
