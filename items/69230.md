## Summary

Keep open. Current `main` still implements phantom `plugins.allow` auditing against installed extension directories plus `listChannelPlugins()` only, while bundled non-channel plugins such as `openai`, `google`, and `acpx` are real manifest-backed plugin IDs. The PR is directionally correct, but the provided review discussion surfaces unresolved correctness and test-isolation concerns, so this is not a conservative close.
