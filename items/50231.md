## Summary

Current `main` already contains the WhatsApp active-listener fix that matches this report: the listener registry is now pinned on `globalThis` so split bundle chunks share one controller map, and the shipped changelog for `v2026.3.22` calls out this exact outbound failure mode.

## Evidence
