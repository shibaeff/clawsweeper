## Summary

Keep open. Current `main` does not implement size-based log rolling: it still caps file logs at `maxFileBytes` and suppresses further writes, and the docs still direct operators to use external log rotation.

## Evidence

- **Current logger suppresses writes instead of rotating:** `buildLogger()` computes `nextBytes`, emits a one-time cap warning when `nextBytes > settings.maxFileBytes`, and returns without any rename/rollover path. Writes still go through `appendLogLine()` -> `fs.appendFileSync()`.
  - file: [src/logging/logger.ts:176](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/logging/logger.ts#L176)
  - command: `nl -ba src/logging/logger.ts | sed -n '150,240p'`
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Docs still describe external rotation as the intended behavior:** The configuration reference says `logging.maxFileBytes` suppresses writes after the cap and explicitly says to use external log rotation for production deployments.
  - file: [docs/gateway/configuration-reference.md:775](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/docs/gateway/configuration-reference.md#L775)

  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **User-facing logging docs do not mention size-based segment rotation:** The logging docs describe the default dated log filename and tailing behavior, but not any in-process `.log.N` rollover; they present the file as the daily rolling path only.
  - file: [docs/gateway/logging.md:20](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/docs/gateway/logging.md#L20)

- **Current tests lock in cap suppression, not rollover:** `log-file-size-cap.test.ts` verifies that after the cap is reached the log size stops growing and only one warning is emitted. There are no corresponding rotation/fd-reopen tests on `main` in `src/logging/*.test.ts`.
  - file: [src/logging/log-file-size-cap.test.ts:52](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/logging/log-file-size-cap.test.ts#L52)

## Risks / Open Questions
