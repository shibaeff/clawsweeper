## Summary

Keep open. Current `main` still contains the exact `existsSync()` then `readFileSync()` sequences in all three targeted security-sensitive paths, so this PR is neither implemented on `main` nor obsolete.

## Evidence

- **Device auth store still has TOCTOU pattern:** `readStore()` checks `fs.existsSync(filePath)` and then separately calls `fs.readFileSync(filePath, "utf8")`, leaving the reported race window intact on current `main`.
  - file: [src/infra/device-auth-store.ts:25](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/infra/device-auth-store.ts#L25)
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Device identity still has TOCTOU pattern:** `loadOrCreateDeviceIdentity()` still gates the read behind `fs.existsSync(filePath)` before `fs.readFileSync(...)` on current `main`.
  - file: [src/infra/device-identity.ts:65](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/infra/device-identity.ts#L65)
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Exec approvals still has both TOCTOU call sites:** Both `readExecApprovalsSnapshot()` and `loadExecApprovals()` still use the two-step existence check followed by a file read on current `main`.
  - file: [src/infra/exec-approvals.ts:432](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/infra/exec-approvals.ts#L432)
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **PR is coherent and targets real current code:** The checked-out `main` matches the report: `rg` finds the `existsSync(filePath)` and `readFileSync(filePath, ...)` pairs in the same three files named by the PR body.
  - command: `rg -n "existsSync\(filePath\)|readFileSync\(filePath" src/infra/device-auth-store.ts src/infra/device-identity.ts src/infra/exec-approvals.ts`
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
