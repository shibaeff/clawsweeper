## Summary

Keep open. Current `main` does not implement Simplified Chinese localization for the macOS native app: the macOS package has no localization resources, the settings/menu UI still uses hardcoded English strings, and the existing Chinese translations are only in the web UI locale file.

## Evidence

- **No macOS localization resources in tree:** A repo search for `*.lproj`, `*.strings`, and `*.xcstrings` under `apps/macos` returned no results, so there is no bundled macOS localization payload for `zh-Hans` or any other locale.
  - command: `find apps/macos -type d -name '*.lproj' -o -name '*.xcstrings' -o -name '*.strings'`
  - sha: [9153598e659d](https://github.com/openclaw/openclaw/commit/9153598e659daf5a52c479731f5b4ab8a5bf16b7)
- **macOS package only copies icon/device resources:** `apps/macos/Package.swift` defines the `OpenClaw` executable target resources as only `Resources/OpenClaw.icns` and `Resources/DeviceModels`; there is no `defaultLocalization` and no localized resource bundle wiring.
  - file: [apps/macos/Package.swift:61](https://github.com/openclaw/openclaw/blob/9153598e659daf5a52c479731f5b4ab8a5bf16b7/apps/macos/Package.swift#L61)
  - command: `nl -ba apps/macos/Package.swift | sed -n '1,220p'`
  - sha: [9153598e659d](https://github.com/openclaw/openclaw/commit/9153598e659daf5a52c479731f5b4ab8a5bf16b7)
- **Requested settings tabs are still hardcoded in English:** The macOS settings root declares tab labels as string literals: `General`, `Channels`, `Voice Wake`, `Config`, `Instances`, `Sessions`, `Cron`, `Skills`, `Permissions`, and `About`, matching the issue's requested surfaces.
  - file: [apps/macos/Sources/OpenClaw/SettingsRootView.swift:26](https://github.com/openclaw/openclaw/blob/9153598e659daf5a52c479731f5b4ab8a5bf16b7/apps/macos/Sources/OpenClaw/SettingsRootView.swift#L26)

- **Other macOS UI copy is also hardcoded English:** Examples include menu items like `Settings…`, `About OpenClaw`, `Quit` and About-panel text such as `GitHub`, `Website`, `Check for updates automatically`, and `Updates unavailable in this build.`
  - file: [apps/macos/Sources/OpenClaw/MenuContentView.swift:149](https://github.com/openclaw/openclaw/blob/9153598e659daf5a52c479731f5b4ab8a5bf16b7/apps/macos/Sources/OpenClaw/MenuContentView.swift#L149)

  - command: `rg -n "String\(localized|NSLocalizedString|LocalizedStringResource|LocalizedStringKey" apps/macos/Sources/OpenClaw apps/shared/OpenClawKit/Sources`
