## Summary

Current `main` already contains code and regression coverage for both parts of #44446: Control UI image attachments are converted into `chat.send` RPC attachments before send, and the chat session/agent selector now disambiguates duplicate per-agent main sessions so the dropdown no longer collapses multiple configured agents into one indistinguishable option.
