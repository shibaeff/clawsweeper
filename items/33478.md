## Summary

Keep open. Current `main` appears to have fixed the thread-context part of the report, but the core feature request is not implemented: `sessions_send` still exposes only `sessionKey`/`label`/`agentId`/`message`/`timeoutSeconds`, and the A2A flow still depends on exact `REPLY_SKIP` / `ANNOUNCE_SKIP` magic strings rather than structured callback parameters or explicit announce routing overrides.

## Evidence
