## Summary

Keep open. Current `main` still has the startup-only `skipAtIfAlreadyRan` guard asymmetry described in the PR, so this is not implemented on `main`. It is also not a safe cleanup close: the PR bundles unrelated fixes, and review discussion already identified a regression in one of those extra hunks.
