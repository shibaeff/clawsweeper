## Summary

Current OpenClaw already handles this class of invalid root-config edit without leaving the gateway down, and it ships schema-path introspection/docs for finding the right key locations. The strict schema rejection remains intentional, but the reported gateway-restart failure is already addressed on main and in release v2026.4.23.
