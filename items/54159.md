## Summary

Keep open. Current `main` still has the blocking overflow cleanup in the shared bounded-response reader, and the regression test from this PR is not present. Main has since consolidated the Matrix wrapper onto that shared helper, so the proposed fix is still relevant but needs a rebase onto current code rather than cleanup closure.
