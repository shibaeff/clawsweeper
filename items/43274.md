## Summary

Current main already suppresses heartbeat guidance for isolated cron agent runs: cron runs use cron-scoped session keys, cron sessions resolve to minimal prompt mode, cron triggers disable heartbeat prompt injection, and minimal prompts omit the Heartbeats section entirely.
