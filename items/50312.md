## Summary

Keep open. Current main has an alternate path that makes many host-local outbound media sends work again under the fs policy, but the specific feature requested here—a configurable global or Feishu/core `mediaLocalRoots` allowlist for outbound sends—is still not implemented on `main`. The underlying March 2026 repro is at least partially obsolete, but the requested configuration knob remains a valid feature request.
