## Summary

Current `main` already has dedicated isolated-cron/subagent follow-up handling for this class of loop: empty or `NO_REPLY` cron replies are no longer treated as interim acknowledgements, descendant subagent completions are awaited and folded into the final result, and stale interim text is suppressed so cron does not re-announce or enqueue duplicate follow-up delivery.

## Evidence

- **Interim/NO_REPLY no longer retried as cron acknowledgements:** `isLikelyInterimCronMessage()` now explicitly returns false for empty text after payload filtering, with a comment calling out `NO_REPLY` and deliberate silence, so silent completion replies are not misclassified as acknowledgements that need another pass.
  - file: [src/cron/isolated-agent/subagent-followup-hints.ts:33](https://github.com/openclaw/openclaw/blob/28fc03c386e14db502429bbe7e062cbe90439ca1/src/cron/isolated-agent/subagent-followup-hints.ts#L33)
  - command: `nl -ba src/cron/isolated-agent/subagent-followup-hints.ts | sed -n '1,80p'`
- **Cron only reruns acknowledgement-only turns when there are no fresh/active descendants:** `run-executor` computes `shouldRetryInterimAck`, then skips the continuation rerun whenever fresh or active descendant subagents exist. That prevents the parent cron turn from re-prompting just because child-completion traffic arrived.
  - file: [src/cron/isolated-agent/run-executor.ts:357](https://github.com/openclaw/openclaw/blob/28fc03c386e14db502429bbe7e062cbe90439ca1/src/cron/isolated-agent/run-executor.ts#L357)

- **Descendant completion handling waits for children and suppresses stale interim output:** `waitForDescendantSubagentSummary()` waits for descendant runs to drain, then only accepts a fresh synthesized reply; if the latest text is still the same interim message or exact `NO_REPLY`, it returns `undefined`. `delivery-dispatch` then suppresses stale interim text and marks delivery attempted so the fallback summary path does not fire a second time.
  - file: [src/cron/isolated-agent/subagent-followup.ts:75](https://github.com/openclaw/openclaw/blob/28fc03c386e14db502429bbe7e062cbe90439ca1/src/cron/isolated-agent/subagent-followup.ts#L75)
