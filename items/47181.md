## Summary

Keep open. Current `main` does not already provide the PR's Azure Claude / Azure AI Foundry onboarding path: there is no `anthropic-azure` auth choice, no matching CLI/env surface, Anthropic docs still describe only direct API key or Claude CLI setup, and the existing bundled Microsoft Foundry plugin is Azure OpenAI-oriented (`AZURE_OPENAI_API_KEY`, `/openai/v1`) rather than Anthropic-on-Foundry.
