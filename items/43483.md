## Summary

Current `main` no longer uses the old blocking TTS delivery path. The `tts` tool now returns synthesized audio as structured tool-result media, and the embedded runner queues and delivers that media without waiting for a bound-channel acknowledgement, which matches the reported isolated-cron hang path being obsolete.
