## Summary

Current `main` does not implement WhatsApp contact-directory syncing. The WhatsApp plugin still exposes a config-backed directory only, outbound target resolution still expects an E.164/group JID or a directory hit, and there is no code path listening for or persisting Baileys `contacts.upsert` data.

## Evidence
