## Summary

Current `main` already implements native Feishu outbound audio for the TTS voice-note path: Feishu is treated as a native voice-note channel, voice-note TTS defaults to Opus, and the Feishu plugin routes Opus/Ogg media as Feishu `audio` messages instead of generic file attachments. This landed after the latest tagged release, so it is implemented on `main` but not yet in `v2026.4.22`.

## Evidence
