## Summary

Keep open. Current `main` does have a shipped skill code-safety scan path, but it does not already include this PR's YARA-style rule set, manifest-validation helpers, trust-verdict API, or secret/unicode-steganography additions. The PR also has unresolved review findings about false positives and manifest/contract handling, so this is not a conservative auto-close candidate.
