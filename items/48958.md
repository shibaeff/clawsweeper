## Summary

Current `main` still passes the full `effectivePrompt` through prompt-image detection, and the bundled memory plugin still prepends recalled transcript snippets inside `<relevant-memories>` blocks. That means the reported phantom-image path is still plausible on `main`, so this PR should stay open; the proposed patch is directionally correct but still has unresolved review feedback on a case-sensitive fast path and a vacuous test.
