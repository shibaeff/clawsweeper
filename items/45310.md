## Summary

Current `main` does not make typing TTL user-configurable. The runtime still defaults `typingTtlMs` to 2 minutes internally, while the exposed config/docs only cover `typingMode` and `typingIntervalSeconds`.

## Evidence

- **Typing TTL remains hard-coded in runtime:** `createTypingController` accepts an internal `typingTtlMs` param, but still defaults it to `2 * 60_000` and uses that timer to stop typing.
  - file: [src/auto-reply/reply/typing.ts:29](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/src/auto-reply/reply/typing.ts#L29)
  - sha: [028b6c9b13c9](https://github.com/openclaw/openclaw/commit/028b6c9b13c960b6f59a7643aff4f6de04e07f71)
- **Reply path does not wire any configurable TTL:** `get-reply` resolves only `typingIntervalSeconds` from agent/session config and passes that into `createTypingController`; it does not read or pass any TTL setting.
  - file: [src/auto-reply/reply/get-reply.ts:231](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/src/auto-reply/reply/get-reply.ts#L231)
  - sha: [028b6c9b13c9](https://github.com/openclaw/openclaw/commit/028b6c9b13c960b6f59a7643aff4f6de04e07f71)
- **Session schema exposes interval, not TTL:** Session config schema includes `typingIntervalSeconds` and `typingMode`, with no `typingTtlMs` or equivalent field.
  - file: [src/config/zod-schema.session.ts:54](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/src/config/zod-schema.session.ts#L54)
  - sha: [028b6c9b13c9](https://github.com/openclaw/openclaw/commit/028b6c9b13c960b6f59a7643aff4f6de04e07f71)
- **Agent defaults schema exposes interval, not TTL:** Agent defaults config likewise exposes `typingIntervalSeconds` and `typingMode`, with no TTL field.
  - file: [src/config/zod-schema.agent-defaults.ts:249](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/src/config/zod-schema.agent-defaults.ts#L249)
  - sha: [028b6c9b13c9](https://github.com/openclaw/openclaw/commit/028b6c9b13c960b6f59a7643aff4f6de04e07f71)
- **Docs only document mode and refresh cadence:** Typing indicator docs describe `typingMode` and `typingIntervalSeconds` as the available tuning knobs and do not mention any configurable TTL.
  - file: [docs/concepts/typing-indicators.md:8](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/docs/concepts/typing-indicators.md#L8)
