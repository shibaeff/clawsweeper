## Summary

Keep open. Current `main` still does not strip bracket-format `[TOOL_CALL]`/`[TOOL_RESULT]` blocks in the heartbeat reply path, so this is not `implemented_on_main` or `cannot_reproduce`. The PR also bundles unrelated Telegram `message:sent` hook wiring that conflicts with current delivery behavior and matches the review comments about duplicate or premature hook emission, so there is not strong evidence for a conservative close.
