## Summary

Current main already resolves isolated cron `payload.model` ahead of session/default selection, persists that resolved model pre-run, and has targeted regression coverage plus CLI docs describing the precedence. The reporter also confirmed the bug no longer reproduces on OpenClaw 2026.4.10.
