## Summary

Keep open. Current `main` does not contain this PR's Discord model-picker fallback: the picker still only dispatches hidden `/model`, waits, rereads the routed session, and emits a mismatch warning if the override is still stale. The acceptance/effective-route return path and regression coverage described in the PR discussion are also absent on `main`.

There is no direct session-store fallback persist in this path.
