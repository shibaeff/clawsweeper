## Summary

Current `main` already fixes the Google Chat auth/fetch problem with a broader implementation than this PR: it stages a dedicated `gaxios` transport with `fetchImplementation`, injects that transport into both GoogleAuth and OAuth2Client flows, and has regression tests covering the no-`window` behavior. The changelog shows this shipped in `v2026.4.22`, so the PR is superseded.
