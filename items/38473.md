## Summary

Current main already routes the Control UI and gateway-reported version through the shared runtime version resolver, which reads nearby package metadata/build-info instead of relying on a stale bundled literal. The same implementation is present in the released v2026.4.22 tag.

## Close Comment

Closing as implemented after a Codex review.
