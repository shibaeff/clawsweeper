## Summary

Keep open. Current `main` does not implement a bulk `openclaw sessions switch-model` flow or any equivalent bulk session-model update command. The CLI exposes only session listing plus `sessions cleanup`, and the session mutation surface in Gateway is single-session `sessions.patch` keyed to one session at a time.
