## Summary

Keep open. Current `main` still has the unfixed hybrid fallback logic in the memory plugin, applies temporal decay/MMR before filtering, and leaves the regression for the `minScore > textWeight` case skipped. The PR discussion shows the author addressed the review concern in follow-up commits, but that refined fix is not present on `main`.

## Evidence
