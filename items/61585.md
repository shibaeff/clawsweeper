## Summary

Current `main` already fixes this by copying bundled `SKILL.md` files into `dist-runtime` during runtime staging instead of symlinking them, which removes the exact `realpath` containment failure described in #61585. The changelog says this shipped in `v2026.4.10`.

## Evidence
