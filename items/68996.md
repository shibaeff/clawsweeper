## Summary

Keep open. The reported Gemma routing bug looks real, but the proposed fix is not yet high-confidence on current `main`: its dynamic-model change can force `api: "google-generative-ai"` onto Gemini CLI-derived templates without also fixing the CLI base URL, and its provider-policy change can rewrite Google `/openai` base URLs even when the effective runtime API still falls back to `openai-completions`. That makes this PR too risky for conservative cleanup.

## Evidence
