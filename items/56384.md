## Summary

Keep open. Current `main` does not contain this PR's Feishu group-ID normalization, and the report remains plausible because the extension still accepts `chat:`/`group:`-style Feishu targets elsewhere while `resolveFeishuGroupConfig()` only matches raw keys.

## Evidence

- **Current resolver still lacks normalization:** `resolveFeishuGroupConfig()` on current `main` only checks an exact key match and then a case-insensitive raw-key match before falling back to `"*"`; it does not strip `chat:`/`group:`/provider prefixes.
  - file: [extensions/feishu/src/policy.ts:128](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/extensions/feishu/src/policy.ts#L128)
  - command: `nl -ba extensions/feishu/src/policy.ts | sed -n '128,149p'`
  - sha: [95b7a85f0607](https://github.com/openclaw/openclaw/commit/95b7a85f0607784d8e33b4ce7e42617174196490)
- **Inbound group lookups use raw event chat_id:** The Feishu bot stores `event.message.chat_id` directly into `ctx.chatId` and passes that raw value into both `resolveFeishuGroupConfig()` and `resolveFeishuReplyPolicy()` for group behavior resolution.
  - file: [extensions/feishu/src/bot.ts:123](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/extensions/feishu/src/bot.ts#L123)
  - command: `nl -ba extensions/feishu/src/bot.ts | sed -n '118,126p;381,386p;471,476p'`
  - sha: [95b7a85f0607](https://github.com/openclaw/openclaw/commit/95b7a85f0607784d8e33b4ce7e42617174196490)
- **Feishu target normalization exists elsewhere:** The extension still has `normalizeFeishuTarget()` which explicitly strips `chat:`, `group:`, and `channel:` prefixes, so prefixed Feishu identifiers remain part of the supported normalization surface in current code.
  - file: [extensions/feishu/src/targets.ts:26](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/extensions/feishu/src/targets.ts#L26)

  - file: [docs/channels/feishu.md:72](https://github.com/openclaw/openclaw/blob/95b7a85f0607784d8e33b4ce7e42617174196490/docs/channels/feishu.md#L72)
