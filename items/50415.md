## Summary

Current `main` does not replay `sessions_history` payloads to Anthropic as structural thinking blocks. The tool returns plain-text JSON, replay sanitization strips tool-result details before model calls, and Anthropic turn validation drops the signed-thinking/tool-use pattern when the only matching tool result is embedded inside user content. The reported HTTP 400 path no longer matches the current code.
