## Summary

Keep open. Current `main` still lacks the PR's stream-dedupe implementation: tool-start commits the full cumulative `chatStream`, and live preview rendering still shows raw `props.stream`, so the reported duplicate-text path around tool interruptions remains plausible on current code.
