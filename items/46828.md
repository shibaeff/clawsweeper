## Summary

Keep open. Current `main` does not implement HTTP/fetch approval gating: `web_fetch` is still wired directly, config only exposes exec/plugin approval families, and gateway/docs only advertise exec/plugin approval methods and events.

## Evidence

- **`web_fetch` has no approval wrapper on main:** `src/agents/openclaw-tools.ts:188-197` creates `webSearchTool` and `webFetchTool` directly via `createWebFetchTool(...)`; there is no HTTP approval gate or wrapper in this path.
  - file: [src/agents/openclaw-tools.ts:193](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/src/agents/openclaw-tools.ts#L193)
  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **Approval config only covers exec/plugin:** `src/config/types.approvals.ts:29-32` defines `ApprovalsConfig` with `exec` and `plugin` only. `src/config/schema.help.ts:661-686` likewise documents approval routing for exec and plugin approvals, with no `approvals.http` or `approvals.httpPolicy` surface.
  - file: [src/config/types.approvals.ts:29](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/src/config/types.approvals.ts#L29)
  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **Gateway API lacks HTTP approval methods/events:** `src/gateway/server-methods-list.ts:33-45` lists `exec.approval.*` and `plugin.approval.*` methods only, and `src/gateway/server-methods-list.ts:165-168` lists only `exec.approval.*` and `plugin.approval.*` events.
  - file: [src/gateway/server-methods-list.ts:33](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/src/gateway/server-methods-list.ts#L33)
  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **Protocol docs still describe only exec and plugin approval families:** `docs/gateway/protocol.md:373-379` documents approval families as exec and plugin only, and `docs/gateway/protocol.md:404-407` lists only exec/plugin approval lifecycle events.
  - file: [docs/gateway/protocol.md:373](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/docs/gateway/protocol.md#L373)
