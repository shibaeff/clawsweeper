## Summary

Keep open. Current `main` does not implement the proposed `messages.outbound.text.emDash` setting, and the existing plugin hook seam is not a full replacement for this PR’s stated scope because mirrored transcript text is computed before `message_sending` hooks run in both direct and gateway send flows. The PR also still has unresolved review-noted correctness bugs in its em-dash rewrite logic, so this is not conservative auto-close material.
