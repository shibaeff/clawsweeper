## Summary

Keep open. Current `main` and the latest release still force Telegram DM answer lanes onto message transport, and the follow-up draft-cleanup / ambiguous-materialize guards discussed on the PR are not present on `main`. This is not implemented on `main`, and the PR still contains substantive unresolved value.
