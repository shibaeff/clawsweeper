## Summary

Keep open. Current `main` still rejects cross-agent targets before checking tree-spawn lineage, which conflicts with the documented `tools.sessions.visibility="tree"` contract for spawned sessions. The PR identifies a real gap, but its proposed reorder is not yet proven safe because `sessions_send` uses the same guard as its access gate and has no separate pre-send A2A authorization check for spawned cross-agent targets.
