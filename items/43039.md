## Summary

Keep open. Current `main` does not include the PR's macOS restart-time plist rewrite, and the existing docs still treat `openclaw gateway install --force` as the repair path when launchd service metadata drifts. The PR also no longer matches the current restart/test shape closely enough to call it obsolete or already implemented.
