## Summary

Keep open. Current `main` still formats successful `config.patch` restart notices as `Gateway restart config-patch ok (config.patch)`, which reads like the restart already completed rather than that a restart is required.

## Evidence

- **Config patch sentinel is created as a successful restart event:** `buildConfigRestartSentinelPayload()` sets `status: "ok"`, and the `config.patch` handler passes `kind: "config-patch"` with `mode: "config.patch"` into that payload builder.
  - file: [src/gateway/server-methods/config.ts:353](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/src/gateway/server-methods/config.ts#L353)
  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **Summary string still says restart already happened:** `summarizeRestartSentinel()` still returns `Gateway restart ${kind} ${status}${mode}`, which for a successful config patch becomes `Gateway restart config-patch ok (config.patch)`.
  - file: [src/infra/restart-sentinel.ts:173](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/src/infra/restart-sentinel.ts#L173)
  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **That formatted message is what gets queued for the post-restart/system notice:** `loadRestartSentinelStartupTask()` calls `formatRestartSentinelMessage(payload)` and uses the result as the outbound/system event text.
  - file: [src/gateway/server-restart-sentinel.ts:316](https://github.com/openclaw/openclaw/blob/68315792677f9628d1370d6869366442f1d3ecad/src/gateway/server-restart-sentinel.ts#L316)
  - sha: [68315792677f](https://github.com/openclaw/openclaw/commit/68315792677f9628d1370d6869366442f1d3ecad)
- **Latest release still has the same summary formatter:** The same `Gateway restart ${kind} ${status}${mode}` implementation is present in release `v2026.4.22` (`00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5`), so there is no shipped fix to point to.
  - file: [src/infra/restart-sentinel.ts:173](https://github.com/openclaw/openclaw/blob/00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5/src/infra/restart-sentinel.ts#L173)

  - sha: [00bd2cf7a376](https://github.com/openclaw/openclaw/commit/00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5)
