## Summary

Keep open. Current `main` does not contain the memory text-search fallback, and the PR still appears to have at least one unresolved behavior regression from review: in the latest patch, the `!hybrid.enabled` branch can still fall back to lexical chunk scanning when vector search returns no hits, which changes existing vector-only semantics.
