## Summary

Current `main` already contains partial equivalents of the requested behavior rules, but not the requested dedicated, cross-model `behavior_principles` system-prompt component. The issue is recent, concrete, and still actionable, so there is not strong evidence for any close reason.

## Evidence

- **Core prompt already covers part of the proposal, but only as compact generic guidance:** `Execution Bias` tells the agent to act in-turn, ask for the one missing blocking decision, retry weak tool results, and provide evidence before finalizing. That overlaps parts of Think Before Acting and Verify Outcomes, but it is not a dedicated behavior-principles section and does not cover the full requested set.
  - file: [src/agents/system-prompt.ts:304](https://github.com/openclaw/openclaw/blob/7f6452897e25543c9263b67aabf140379d1c0b0c/src/agents/system-prompt.ts#L304)
  - sha: [7f6452897e25](https://github.com/openclaw/openclaw/commit/7f6452897e25543c9263b67aabf140379d1c0b0c)
- **Richer behavior guidance exists today, but it is GPT-5-only:** The GPT-5 overlay adds execution-policy, tool-discipline, and completion-contract rules such as asking one concise blocking question, resolving prerequisites, using the smallest meaningful verification step, and checking requirements before finalizing. The same file only returns that overlay when `isGpt5ModelId(modelId)` is true, so it is not a cross-model implementation of this issue's request.
  - file: [src/agents/gpt5-prompt-overlay.ts:63](https://github.com/openclaw/openclaw/blob/7f6452897e25543c9263b67aabf140379d1c0b0c/src/agents/gpt5-prompt-overlay.ts#L63)

- **Docs describe the current design as a small core prompt plus model-family-specific overlays:** The public system-prompt docs say provider plugins can replace only named sections or inject prompt prefixes/suffixes, and specifically describe GPT-5 guidance as model-specific tuning. That matches the current architecture and cuts against treating this issue as already implemented for all models.
  - file: [docs/concepts/system-prompt.md:13](https://github.com/openclaw/openclaw/blob/7f6452897e25543c9263b67aabf140379d1c0b0c/docs/concepts/system-prompt.md#L13)
