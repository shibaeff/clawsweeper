## Summary

Current `main` already addresses the reported WebChat clutter: inbound envelope/system metadata is stripped from visible chat text, and Feishu inbound previews are no longer queued as `System:` transcript events. That resolves the concrete UI problem even though the implementation chose removal/display-normalization rather than a collapse toggle.
