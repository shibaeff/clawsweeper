## Summary

Keep open: current `main` does not contain this fix. `src/agents/tools/cron-tool.ts` still strips `:thread:` suffixes and infers delivery from the parent peer only, while Telegram DM routing on `main` still produces scoped DM thread session keys like `agent:main:telegram:direct:<chatId>:thread:<chatId>:<threadId>`. The regression test proposed by the PR is also absent.
