## Summary

Keep open. Current `main` does not implement the PR’s `/learn` command or its handler wiring, so this is not `implemented_on_main`. The repo does have adjacent memory infrastructure already shipped: automatic pre-compaction memory flush and `before_reset` hooks. That makes the PR additive rather than obsolete, and there is not strong evidence for `cannot_reproduce`, `clawhub`, or `incoherent`.
