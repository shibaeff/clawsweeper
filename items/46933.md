## Summary

Keep open. Current `main` still renders `Last heartbeat` through `buildStatusLastHeartbeatValue` in `src/commands/status.command-sections.ts`, and that code still produces the reported `ago ago`/`unknown` output on the live `status --deep` path. The PR’s original edit target has been refactored since March, but the underlying bug remains in current code, so there is not strong evidence to close this PR.
