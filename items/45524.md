## Summary

Current `main` does not implement this PR’s core startup warning. Gateway startup still only warns from `collectEnabledInsecureOrDangerousFlags`, and that collector does not flag explicit `agents.defaults.sandbox.mode="off"`. Related security-audit checks exist, but they are separate from startup warnings and cover narrower/different conditions.
