## Summary

Keep open. Current `main` still exposes only `message channel info|list` in the Discord admin CLI surface, while the Discord action layer and capability discovery already support `channel-create`, `channel-edit`, `channel-delete`, and `channel-move`. The PR still addresses a real wiring gap, and the review-discussion fixes called out on the PR were already followed up in later PR commits.
