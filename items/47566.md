## Summary

Keep open. Current `main` still does not implement a `openclaw plugins init` scaffold flow, and the docs still require manual plugin bootstrapping. The PR also has an unresolved design mismatch from review: its scaffold validator rejects scoped ids even though current install validation explicitly supports `@scope/name` plugin ids.
