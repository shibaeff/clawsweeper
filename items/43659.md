## Summary

Keep open. Current `main` still uses a single shared Copilot token cache entry with no profile/PAT binding, while the embedded runner still rotates auth profiles on rate-limit failures. The PR addresses a real gap that is not implemented on `main`; it likely needs a rebase because the code now lives under `src/agents/` instead of the PR's older `src/providers/` path.

## Evidence
