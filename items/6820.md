## Summary

Keep open. Current `main` still does not grant `xhigh` to `openai-codex/gpt-5.2`. The issue body points at an old generated file path, but the equivalent gating now lives in the OpenAI Codex provider plugin and still omits the base `gpt-5.2` model.
