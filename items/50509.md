## Summary

Current main already qualifies slash-containing provider-native model IDs in the Control UI before calling `sessions.patch`, and current regression coverage explicitly covers nested model IDs. The reported Ollama dropdown failure is already implemented/fixed in the shipped codebase.

## Evidence

- **Control UI builds provider-qualified picker values:** `buildChatModelOptionFromLookup()` uses `buildQualifiedChatModelValue(entry.id, provider)`, so a provider-native model ID containing slashes is emitted as `provider/model-id` rather than a bare nested ID.
  - file: [ui/src/ui/chat-model-ref.ts:266](https://github.com/openclaw/openclaw/blob/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f/ui/src/ui/chat-model-ref.ts#L266)
  - sha: [9b7f6250f43b](https://github.com/openclaw/openclaw/commit/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f)
- **Chat header forwards the qualified value unchanged:** `switchChatModel()` sends the selected picker value directly to `sessions.patch`, so once the UI value is qualified it no longer hits the bare-ID parsing failure described in the issue.
  - file: [ui/src/ui/chat/session-controls.ts:234](https://github.com/openclaw/openclaw/blob/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f/ui/src/ui/chat/session-controls.ts#L234)
  - sha: [9b7f6250f43b](https://github.com/openclaw/openclaw/commit/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f)
- **UI regression tests cover slash-containing model IDs:** Current tests expect provider-native IDs with slashes to stay qualified, including cases like `openrouter/google/gemma-4-26b-a4b-it` and `nvidia/moonshotai/kimi-k2.5`.
  - file: [ui/src/ui/chat-model-ref.test.ts:38](https://github.com/openclaw/openclaw/blob/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f/ui/src/ui/chat-model-ref.test.ts#L38)
  - sha: [9b7f6250f43b](https://github.com/openclaw/openclaw/commit/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f)
- **Gateway integration test covers `sessions.patch` with nested IDs:** The gateway test patches `model: "nvidia/moonshotai/kimi-k2.5"` and asserts the stored session override is `providerOverride: "nvidia"` plus `modelOverride: "moonshotai/kimi-k2.5"`.
  - file: [src/gateway/server.sessions.gateway-server-sessions-a.test.ts:1602](https://github.com/openclaw/openclaw/blob/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f/src/gateway/server.sessions.gateway-server-sessions-a.test.ts#L1602)
