## Summary

Keep open. Current `main` still appears to have the sidebar copy-button bug the PR describes: markdown rendering inserts `.code-block-copy` buttons, but the delegated click handler is attached to `.chat-thread` only, while the sidebar markdown is rendered under sibling `.chat-sidebar` markup with no corresponding click binding.
