## Summary

Keep open. Current `main` still accepts and concatenates hook-injected prompt strings without any length cap, then injects them directly into the user/system prompt path. This PR appears to address a real gap, and the follow-up commits in the discussion already addressed the review comments about truncation math, warning wording, legacy-path tests, and the combined legacy+phase merge point.
