## Summary

Keep open. Current `main` only partially addresses the report: generic `message` schema no longer exposes `components`, and zero-valued poll defaults no longer trip the send path, but poll creation fields are still part of the shared message-tool schema and `action="send"` still hard-errors when non-zero/true poll params are present.
