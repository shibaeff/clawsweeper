## Summary

Current `main` already removes the synchronous startup stall by deferring model-pricing bootstrap to a microtask, but it does not implement the PR's proposed `models.pricing.enabled` opt-out. The PR is therefore partially superseded, not implemented on `main`, and still represents a distinct behavior change that could be useful for offline or restricted-egress setups.
