## Summary

Keep open. Current `main` still omits `ctx` when the embedded runner sends tools through `splitSdkTools() -> toToolDefinitions() -> runBeforeToolCallHook()`, even though the hook contract and the client-tool path both carry `agentId`/session metadata. The PR is addressing a real gap, and the requested regression test coverage is also not present on `main` yet.

## Evidence
