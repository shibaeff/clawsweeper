## Summary

Keep open. Current `main` still rejects `api: "google-vertex"` in config/schema even though runtime and docs already contain partial `google-vertex` support. The PR’s 3-line enum change also appears incomplete on current `main`, because Gemini/Google transcript safeguards still key off `isGoogleModelApi`, which does not include `google-vertex`.
