## Summary

Keep open. Current `main` still rejects `sessions_spawn` calls that include `streamTo` or `resumeSessionId` when runtime is `subagent`, and the shipped tests and docs still codify that behavior. This PR is not implemented on `main`, and the reported bug is still plausible/current rather than obsolete.
