## Summary

Keep open. Current `main` still has a strict Anthropic SSE `JSON.parse` hot path and eager streamed tool-call JSON reparsing in the same transport area implicated by the report. Webchat/gateway now have explicit chat-error cleanup paths, but I did not find a regression test or code change that proves the non-ASCII `edit`-tool failure is fixed end-to-end.
