## Summary

Keep open. Current `main` still drops mention-gated group messages before hook dispatch on several affected channels, so the reported plugin-observability gap is still real. There is a partial new opt-in path on `main` for Telegram and Signal via `groups.*.ingest`, but there is no generic early inbound hook and the issue still reproduces in Discord, Slack, LINE, iMessage, and BlueBubbles.
