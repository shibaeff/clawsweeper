## Summary

Keep open. Current `main` still records missing native Ollama counters as zero instead of estimating usage, so the PR's main behavior is not implemented. That still matters because usage surfaces in the dashboard/UI depend on positive token totals. The PR discussion also surfaced valid follow-up details around preserving explicit zero counts and image payload estimation, which reinforces that this is still active review material rather than stale-close material.
