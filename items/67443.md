## Summary

Keep open. Current `main` does not already contain this PR’s behavior, and the PR discussion surfaces two credible P1 regressions in the proposed patch: it would drop default-session context-token fallback/override handling and could reject `openclaw/main` when `agents.list` is omitted. That is not conservative cleanup material.
