## Summary

Keep open. Current `main` still implements memory recall as prompt/tool guidance, not as a forced pre-response runtime recall step, and there is no matching `memorySearch.recallBeforeResponse`-style config or enforcement path on `41c5ffc5d5f4dc0f1df6f2c9a7efd8ce6a627f00`.
