## Summary

Keep open. Current `main` at `028b6c9b13c960b6f59a7643aff4f6de04e07f71` does not already contain the cron lifecycle hook feature from PR #45465. The hook config/job fields are absent, the config and gateway schemas do not admit them, and the cron timer path still executes jobs directly with no lifecycle hook dispatch.

## Evidence

- **Cron config has no hook surface:** `CronConfig` currently ends with failure alert/destination fields; there is no `hooks` field and no hook entry types in this file.
  - file: [src/config/types.cron.ts:30](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/src/config/types.cron.ts#L30)
- **Cron jobs do not support per-job hooks:** `CronJob` currently adds only `state`; there is no `hooks` property on stored jobs.
  - file: [src/cron/types.ts:168](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/src/cron/types.ts#L168)
- **Config validation would reject `cron.hooks`:** The strict `cron` object schema lists enabled/store/retry/webhook/sessionRetention/runLog/failureAlert/failureDestination only; no `hooks` key is accepted.
  - file: [src/config/zod-schema.ts:530](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/src/config/zod-schema.ts#L530)
- **Gateway protocol does not expose job hooks:** `CronJobSchema` and `CronJobPatchSchema` do not include a `hooks` field, so the gateway contract on `main` does not support this PR's surface.
  - file: [src/gateway/protocol/schema/cron.ts:262](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/src/gateway/protocol/schema/cron.ts#L262)
- **Scheduler still runs jobs without hook dispatch:** `runDueJob()` marks the job started and then calls `executeJobCoreWithTimeout(state, job)` directly; there is no `loadHookEntries`/`runCronHooks` lifecycle handling around execution.
  - file: [src/cron/service/timer.ts:739](https://github.com/openclaw/openclaw/blob/028b6c9b13c960b6f59a7643aff4f6de04e07f71/src/cron/service/timer.ts#L739)
- **Hook runner files are absent on main:** A filesystem check for `src/cron/hooks.ts` and `src/cron/hooks.test.ts` returned no matches, consistent with the feature not being present in the checked-out branch.
  - command: `find src/cron -maxdepth 1 -name 'hooks.ts' -o -name 'hooks.test.ts' | sort`
