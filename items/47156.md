## Summary

Keep open. Current `main` only partially overlaps this PR: Control UI localization exists and `ui/src/i18n/locales/zh-CN.ts` already contains several of the added Chinese keys, but the PR's README localization is not present on `main`. `Chinese-README.md` does not exist in the checkout, and `README.md` has no Chinese/English language-switcher link. The repo's current localization guidance also says non-English UI locale bundles are generated output, so this PR would need rework rather than a straight merge, but none of the allowed close reasons is strongly supported.
