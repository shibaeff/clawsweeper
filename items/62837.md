## Summary

Keep open. Current `main` still gates WhatsApp group replies off `extractMentionedJids`, and I did not find strong current-main proof that the WhatsApp MD/group-wrapper payload from this report is now covered end to end. The code and docs still say explicit WhatsApp mentions should wake the bot, so there is not enough evidence to close as fixed or obsolete.
