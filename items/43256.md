## Summary

Keep open. Current `main` still contains the exact broad system-prompt line reported in the issue, and the clarification proposed in PR #43280 does not appear to have landed. The underlying Telegram attachment path exists on `main`, but the prompt wording that can cause GPT-5.4 to refuse it is still present.
