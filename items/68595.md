## Summary

Current main already guards against the reported REM metadata contamination path: contaminated dream-summary snippets are filtered before they enter or rank in short-term promotion, managed dreaming blocks are stripped before daily-note re-ingestion, and concept-tag extraction tests cover filename noise. The same implementation is present in release v2026.4.23.
