## Summary

Keep open. Current `main` still documents and validates `ui.seamColor`, but the Control UI bootstrap contract, bootstrap response, and frontend bootstrap loader do not carry or apply that value, so the reported behavior remains plausible. The PR also has unresolved review evidence of a build-breaking syntax error and broader CSS-token side effects, so there is not strong evidence to close it as fixed or obsolete.
