## Summary

Keep open. Current `main` still appears to allow provider auth env vars to reach host `exec` child processes, so the underlying security report is not obsolete. The PR also should not be auto-closed as implemented, because its proposed fix is not present on `main` and the PR discussion already identified an override-path gap in the branch head.
