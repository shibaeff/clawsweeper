## Summary

Keep open. Current `main` still hardcodes the bare `/new`/`/reset` startup message, tests still lock that wording in, and the shipped config/docs only expose `agents.defaults.startupContext.*` controls for preloading daily memory, not a configurable `session.resetPrompt` override.
