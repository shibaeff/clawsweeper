## Summary

Current `main` does not include the null-byte rejection proposed by PR #50521, so there is no strong basis to close it as already implemented or obsolete. The archive path validator still accepts strings containing `\0` through its explicit checks, and the dedicated regression tests from the PR are not present on `main`.

## Evidence
