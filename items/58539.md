## Summary

Current `main` already contains the live-model-switch gating and retry behavior that prevents isolated subagent runs with explicit model overrides from failing immediately with `LiveSessionModelSwitchError`. The changelog records this as shipped in `v2026.3.31`.

## Evidence

- **Shipped fix noted in changelog:** `CHANGELOG.md:1902` under `## 2026.3.31` says transient cron and subagent model overrides are no longer misread as persisted live-session switches, so isolated runs no longer fail with `LiveSessionModelSwitchError`.
  - file: [CHANGELOG.md:1902](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/CHANGELOG.md#L1902)
  - command: `nl -ba CHANGELOG.md | sed -n '1888,1912p'`
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Current live-switch gate is explicit and pending-flag based:** `src/agents/live-model-switch.ts:123` now only returns a switch when `liveModelSwitchPending` is set and the persisted selection actually differs, and clears stale pending state when the current run already matches.
  - file: [src/agents/live-model-switch.ts:123](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/agents/live-model-switch.ts#L123)
  - command: `nl -ba src/agents/live-model-switch.ts | sed -n '121,190p'`
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Runner only throws on an actual pending live switch:** `src/agents/pi-embedded-runner/run.ts:974` calls `shouldSwitchToLiveModel(...)` and throws `LiveSessionModelSwitchError` only for a requested pending switch at a safe restart boundary, replacing the older unconditional persisted-selection comparison described in the report.
  - file: [src/agents/pi-embedded-runner/run.ts:974](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/agents/pi-embedded-runner/run.ts#L974)
  - command: `nl -ba src/agents/pi-embedded-runner/run.ts | sed -n '920,1045p'`
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Retry behavior is covered on current main:** `src/auto-reply/reply/agent-runner-execution.test.ts:2155` verifies the active prompt restarts after `LiveSessionModelSwitchError`, and `CHANGELOG.md:1408` records follow-up subagent retry handling around session override state.
  - file: [src/auto-reply/reply/agent-runner-execution.test.ts:2155](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/auto-reply/reply/agent-runner-execution.test.ts#L2155)
