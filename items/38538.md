## Summary

Keep open. Current `main` does not implement `agents.*.memorySearch.remote.proxy`, and the memory remote HTTP path still lacks the proxy-mode/dispatcher plumbing needed to make Gemini embedding requests honor a proxy. That rules out `implemented_on_main` and `cannot_reproduce`, and this remains OpenClaw core scope rather than ClawHub.
