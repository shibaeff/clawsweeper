## Summary

Keep open. Current `main` has automatic subagent bootstrap/context handling, but it does not implement this PR's public `contextFiles` / `inheritContext` API, and cross-agent spawns still intentionally resolve the child workspace from the target agent rather than inheriting the requester workspace. The PR also still has unresolved review blockers in the submitted diff.
