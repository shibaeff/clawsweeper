## Summary

Keep open. I found no high-confidence evidence that `main` fixes the reported `/subagents list` regression from `v2026.4.23`. The current slash-command path still populates list output from `listControlledSubagentRuns(requesterKey)`, and the inspected post-release diffs do not change the command/control wiring the report points at. Current docs and tests still describe `/subagents list` as showing live children and recently ended runs, so the reported empty output remains plausible on `main` and should stay open for maintainer investigation.
