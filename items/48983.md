## Summary

Keep open. Current `main` still hard-exits from `scripts/install.ps1` during the documented PowerShell `iex`/scriptblock install flows, and there is no equivalent fix or regression test on `main`. The linked issue remains materially valid even though its exact function-level repro text is a bit stale, and PR #48983 appears to address the gap more completely than the earlier related PR #41931.
