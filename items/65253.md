## Summary

Close as `clawhub`. The PR proposes a useful optional image-generation capability, but current `main` documents that this kind of functionality should ship as an external plugin/provider via ClawHub or npm, and `extensions/image-generation-core` is presently a private manifest-less runtime support package rather than a user-facing bundled plugin surface.

## Evidence
