## Summary

Current `main` already contains Anthropic-thinking replay/compaction protections that address this failure mode: signed thinking turns are no longer mutated during replay sanitization, Anthropic turns are revalidated before embedded runs and compaction, and truncation repair is rerun after compaction limits history. The relevant implementation is present on `main` in commit `4f4d2ef1dffd27fc7486850185af16429061606d`, but it does not appear to be part of the latest tagged release `v2026.4.22`.
