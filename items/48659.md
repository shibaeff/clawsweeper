## Summary

Keep open. Current `main` only contains a subset of this PR's stack. One attachment-ingress fix from the branch is already present, but the core PR claims are still unshipped: MS Teams channel replies still default to `top-level` when `requireMention=false`, there is no `channel_deleted` hook or `extensions/msteams-channel-archive` plugin on `main`, and the Teams conversation store still does not persist the readable recent-channel-focus metadata described in the PR. The review thread also already surfaced unresolved integration/rebase concerns around the proposed `resolveGraphToken` API change, so this needs normal maintainer/author follow-up rather than conservative auto-close.
