## Summary

Keep open. Current `main` at `0bef73d151bd0a8d547b28bf16c3aa2e21078e0f` still does not expose `api.runtime.gateway.dispatchMethod`, so this PR is neither implemented nor obsolete. The plugin runtime contract and gateway plugin loader only support the existing narrower late-bound surfaces (`subagent` and `nodes`), and the docs still describe those surfaces rather than a generic gateway-dispatch API.
