## Summary

Keep open. Current `main` only supports one fixed `heartbeat.every` interval plus optional `activeHours` gating; the requested time-of-day schedule/daypart variable intervals are not implemented on `main` yet.

## Evidence

  - file: [src/config/zod-schema.agent-runtime.ts:19](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/config/zod-schema.agent-runtime.ts#L19)
