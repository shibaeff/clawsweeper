## Summary

Current `main` and the latest release still hard-ban using the message tool in group-chat context, and local tests assert that exact prohibition. The PR's core change is therefore not implemented on `main`. The review discussion also improved the wording from vague "non-text content" to concrete attachment/media types, so this PR still contains potentially useful work rather than a stale duplicate.
