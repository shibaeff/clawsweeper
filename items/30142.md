## Summary

Keep open. Current `main` still routes structured replies through adapter-level `sendPayload` when available, but Google Chat, Mattermost, MS Teams, and Synology Chat still do not implement `sendPayload` on `main`. The PR’s exact diff is stale against current adapter structure, yet the underlying gap is not already implemented, so there is not enough evidence to close it conservatively.
