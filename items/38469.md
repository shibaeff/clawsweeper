## Summary

Current main already addresses this class of leak: the diffs plugin injects its guidance into system-prompt space via `prependSystemContext`, and the macOS chat pipeline strips AI-only inbound metadata from user messages before rendering. The same implementation is present in the shipped `v2026.4.22` release commit.

## Evidence
