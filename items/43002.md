## Summary

Current `main` already implements the requested automated memory consolidation via the bundled `memory-core` Dreaming system: scheduled background sweeps ingest short-term signals, rank candidates with recency/consolidation heuristics, and promote qualified entries into `MEMORY.md`. The feature is documented, wired into CLI/config, and is already present in the `v2026.4.22` release tag.
