## Summary

Keep open. Current `main` still has a 30s TUI streaming watchdog, but the only configurability is an internal `streamingWatchdogMs` handler context field. The normal TUI wiring does not pass any user/config value, and the public config/docs surfaces expose ACP coalescing and LLM idle timeout settings, not a TUI streaming-watchdog threshold. The report is recent, coherent, and corroborated by multiple users in the issue discussion.

## Evidence
