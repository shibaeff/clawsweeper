## Summary

Keep open. Current `main` has enabling primitives for external voice nodes, but it does not ship Brabble or any equivalent distributed/headless voice-wake node implementation. The request is still an unmet feature/architecture item, and it does not cleanly fit the `clawhub` close bucket because nodes/platforms are still first-class repo surfaces.

## Evidence
