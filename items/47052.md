## Summary

Current main already separates trusted inbound context from untrusted channel content, neutralizes spoofed system markers, wraps channel metadata as external untrusted content, and blocks cross-agent messaging unless `tools.agentToAgent` is explicitly enabled and allowlisted. That covers the core request well enough.
