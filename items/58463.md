## Summary

Close as `clawhub`: current OpenClaw scope and plugin docs point this kind of Claude Code bridge to an external plugin/ClawHub package rather than a new bundled core extension, and main already has supported Claude Code integration paths through ACP sessions and the `claude-cli` backend.
