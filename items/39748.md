## Summary

Keep open. Current `main` still leaves Nostr relay config as an unconstrained string array, the setup paths still allow any `ws://` relay, and runtime uses configured relays directly. The PR addresses a live gap and the discussion improved it, but it needs a refresh against the current tree rather than an auto-close.
