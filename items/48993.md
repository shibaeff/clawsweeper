## Summary

Keep open. Current `main` still intentionally flattens remote markdown images to alt text, and newer Control UI CSP/docs/tests explicitly prohibit remote `http(s)` image fetches. This PR has useful implementation work, but it conflicts with the current privacy/networking model and needs explicit maintainer product/security judgment rather than conservative auto-close.
