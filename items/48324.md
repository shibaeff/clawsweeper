## Summary

Keep open. Current `main` still refreshes skill snapshots only on watcher-version and normalized `skillFilter` changes, while skill eligibility is computed from additional runtime inputs (bins, env, config, remote state). The PR discussion shows the author addressed the first round of review feedback, but current shipped code still does not prove this bug is fixed end-to-end across the affected snapshot caches.
