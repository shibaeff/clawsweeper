## Summary

Keep open. Current `main` does not implement ACP skill-context injection for `sessions_spawn(runtime="acp")`: the tool schema and docs expose no `skills`/`auto`/default-skill option, and the ACP session initialization API has no field to carry skill context into the spawned harness session.

## Evidence

- **`sessions_spawn` ACP schema has no skill-injection field:** The tool schema exposes `task`, `label`, `runtime`, `agentId`, `resumeSessionId`, `model`, `cwd`, `thread`, `mode`, `sandbox`, `context`, `streamTo`, `lightContext`, and attachments, but no `skills`, `skillFilter`, or similar ACP skill-context parameter.
