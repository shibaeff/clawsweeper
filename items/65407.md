## Summary

Current `main` already restores CLI-runner prompt hooks. The shared CLI preparation path now invokes the global hook runner for `before_prompt_build` and legacy `before_agent_start`, applies prompt/system-prompt mutations, and has targeted regression tests. The fix was shipped in `v2026.4.23`.

## Evidence

- **CLI runner now executes prompt hooks during preparation:** `prepareCliRunContext` gets the global hook runner, checks `before_prompt_build` / `before_agent_start`, loads CLI session history, passes trigger/channel/model context into `resolvePromptBuildHookResult`, prepends hook prompt context, composes `prependSystemContext` / `appendSystemContext`, and returns the mutated prompt in `context.params`.
  - file: [src/agents/cli-runner/prepare.ts:310](https://github.com/openclaw/openclaw/blob/1bdf5307d99d9c1edcbe099d7c2fc3947a9b0a19/src/agents/cli-runner/prepare.ts#L310)
