## Summary

Keep open. Current `main` still misses parts of the requested subagent docs cleanup, and the PR discussion shows the proposed patch also needs revision before it is safe to merge: `streamTo` is ACP-only in code, the default denied-tool description is broader than the PR text, and current docs still underdocument deeper nesting/config details.
