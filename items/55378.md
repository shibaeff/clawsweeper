## Summary

Keep open. Current `main` clearly fixes commentary-phased reply leaks, but I did not find high-confidence evidence that the Anthropic Claude Opus 4.6 path described here is fixed. The remaining delivery suppression is still phase-driven, while the Anthropic transport still streams plain text/tool-use blocks without phase metadata.

## Evidence

- **Delivery suppression is phase-gated:** User-visible assistant output is suppressed only when `resolveAssistantMessagePhase(...) === "commentary"`, and streaming updates likewise early-return only for `deliveryPhase === "commentary"`. Unphased assistant text still flows through normal block-reply delivery.

If a provider emits plain text blocks without that metadata, the message does not resolve to commentary.
