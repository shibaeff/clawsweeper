## Summary

Current `main` and the shipped `v2026.4.23` release already contain the loopback CDP proxy-bypass implementation this issue asks for: loopback HTTP probes run under a scoped `NO_PROXY` bypass, loopback WebSocket handshakes use direct agents, and regression tests cover both behaviors.
