## Summary

Current main already contains a targeted fix for main-session `systemEvent` cron jobs: cron now forces heartbeat delivery via `target: "last"` for these jobs and the gateway cron adapter preserves that override when invoking the heartbeat runner. New regression tests cover both the cron timer path and the gateway adapter path.
