## Summary

Keep open. Current `main` still resolves Telegram topic config by exact `message_thread_id` only, so the PR's `topics."*"` fallback is not implemented in the shipped code, docs, or tests.

## Evidence

That means wildcard topic defaults are not implemented on current `main`.
