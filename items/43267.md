## Summary

Issue #43267 is about `memory-openviking`/OpenViking behavior, but current `main` does not contain that plugin or any `openviking` integration. OpenClaw’s scope docs say optional capabilities should usually live as plugins maintained in their own repos and discovered via ClawHub/npm, so this belongs with the external plugin/service rather than OpenClaw core.
