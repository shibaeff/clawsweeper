## Summary

Keep open. Current `main` still ships `@mariozechner/pi-agent-core` 0.70.0, and that package's published `dist/agent-loop.js` still calls `.filter()` directly on `message.content` / `assistantMessage.content` without an `Array.isArray` guard. OpenClaw also still has local code that explicitly treats assistant `content` as `string | array`, so the report is neither implemented on `main` nor obsolete.
