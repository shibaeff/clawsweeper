## Summary

Current `main` already implements the requested mid-run `/steer` flow: `/steer` is a built-in slash command (alias `/tell`), it resolves current-session or subagent targets, injects the message into the active run with `chat.send`/`deliver: false`, returns a short acknowledgement, and has docs plus regression coverage. The same implementation is present in release `v2026.4.22`.
