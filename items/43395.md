## Summary

Current `main` no longer routes the default single-agent workspace to `~/.openclaw/workspace-main`. The default path resolves to `~/.openclaw/workspace`, and both current docs and tests match that behavior, so the March 2026 report no longer matches current OpenClaw behavior.
