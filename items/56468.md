## Summary

Keep open. Current `main` still lacks the proposed in-process clobbered-backup dedup and still lacks the wildcard plugin-API semver handling from this PR, so this is not implemented on `main`. The PR also has unresolved review concerns and mixes two distinct fixes, which makes it poor auto-close material for conservative cleanup.
