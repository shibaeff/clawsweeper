## Summary

Keep open. Current main has migration docs and a first-class `openclaw backup` archive flow, but that flow explicitly backs up full state, auth profiles, credentials, sessions, and optionally workspaces. It does not implement the issue's proposed safe/shareable agent package that excludes runtime state and secrets, and the CLI surface only exposes `backup create` and `backup verify`, not an export/import or pull/run package workflow.
