## Summary

Keep open. Current `main` at `028b6c9b13c960b6f59a7643aff4f6de04e07f71` still routes Slack `message_changed` and `message_deleted` subtypes into system events, and there is no `channels.slack.ignoreSubtypes` config seam in the current Slack channel config types.
