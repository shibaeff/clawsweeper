## Summary

Keep open. Current `main` still lacks the requested direct session-deletion CLI. The only registered `sessions` subcommand is `cleanup`, which performs policy-driven maintenance; there is no `sessions rm`, `sessions clear`, or `--older-than` path in the checked-out code, docs, or local git history I inspected.
