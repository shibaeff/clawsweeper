## Summary

Keep open. Current `main` still has the unresolved pieces from the report: local `node-llama-cpp` embedding batches are still concurrent by default, there is still no exposed CPU-only/Metal toggle for local embeddings, and the docs still do not provide Apple Silicon Metal/GPU guidance. Existing crash-handling coverage only addresses rejected JS promises, not native library aborts that can take down the gateway.

## Evidence
