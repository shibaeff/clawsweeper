## Summary

Keep open. Current `main` still classifies plain `Error("acpx exited with code 1")` turn failures as `ACP_TURN_FAILED`, so the PR's core behavior is not implemented on `main` yet. The author's follow-up comments addressed earlier review concerns, but those fixes are only in the PR discussion/branch evidence, not in the checked-out `main` code.
