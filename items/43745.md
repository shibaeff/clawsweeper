## Summary

Current `main` already addresses this: bundled plugin skill paths that point into `node_modules` are copied into a dist-owned `bundled-skills/` directory and the generated manifest is rewritten to use that in-root path, which removes the installed-extension false-positive root-escape condition while keeping the runtime escape checks in place.
