## Summary

Keep open. Current `main` already repairs dangling assistant tool calls more broadly than this PR's last-message fix, but it still does not implement the PR's friendly handling for raw Anthropic `tool_use`/`tool_result` mismatch errors. The PR's core user-facing change is therefore not implemented on `main` yet.
