## Summary

Keep open. Current `main` still does not implement a Talk Mode idle timeout or auto-deactivation after Voice Wake. The Talk config/schema only expose `interruptOnSpeech` and `silenceTimeoutMs`, macOS Voice Wake still enables Talk Mode directly, and macOS Talk Mode resumes listening after each turn until the user explicitly exits it.
