## Summary

Current `main` still does not automatically relay an ACP one-shot child's final successful answer back into the parent chat. The implemented paths only stream progress snippets and/or emit a generic completion status unless a special `terminalSummary` is present, and ACP currently only derives that summary for blocked/permission-style outcomes.

## Evidence

- **Parent stream relay ends with generic completion text:** The ACP parent-stream relay emits progress snippets during assistant deltas, then only `"<agent> run completed"` / `"completed in Ns"` on lifecycle end. It does not capture or forward the child's final visible answer payload.
