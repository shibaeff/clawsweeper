## Summary

Keep open. Current `main` has added the prerequisite `availableTools` plumbing for context engines, but it still does not implement the PR's proposed `toolsOverride` return field or any runner path that selectively injects tool schemas per turn. The PR discussion also surfaces an unresolved contract question around whether `toolsOverride: []` should suppress all schemas or act as a no-op.
