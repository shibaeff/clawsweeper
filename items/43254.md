## Summary

Keep open. Current `main` and the latest release still key Control UI session tokens by the exact parsed gateway host, so `127.0.0.1`, `localhost`, and `[::1]` remain separate scopes. This PR addresses a plausible real bug and is not implemented on `main`, although the review comment about overbroad `127.*` hostname matching is a real concern that likely needs follow-up before merge.
