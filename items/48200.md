## Summary

Keep open. Current `main` has the narrower iMessage self-chat echo work from #38440, but it does not contain this PR's `accountHandle` config surface, startup misconfiguration validation, or the per-message sender-vs-own-handle fallback. The remaining inbound logic still relies on `is_from_me`, self-chat detection, and existing echo caches, so this PR is not implemented on `main` and is not safe to close as obsolete.
