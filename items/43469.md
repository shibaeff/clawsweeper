## Summary

Keep open. Current `main` still excludes markdown from `src/security/skill-scanner.ts`, so the reported `SKILL.md` blind spot remains in the installed-skill and deep-audit code-safety paths. The PR is directionally valid, but the review discussion identifies unresolved bypasses in its new critical markdown regex, so this is not strong evidence for a conservative closure.

## Evidence
