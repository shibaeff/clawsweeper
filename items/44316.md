## Summary

Current `main` does not implement this PR. The web UI still keeps tool/toolResult turns visible when `showToolCalls` is enabled, still renders pure tool-call assistant turns as chat items, and still lacks the proposed `SECURITY NOTICE` / `EXTERNAL_UNTRUSTED_CONTENT` stripping in message extraction.
