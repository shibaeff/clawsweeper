## Summary

Keep open. Current `main` still exposes `streamTo` in the `sessions_spawn` tool schema for all runtimes, then explicitly rejects it for non-ACP spawns, which matches the reported subagent failure mode. The current test suite also locks in that rejection behavior, so there is no strong evidence this issue is implemented on `main`.
