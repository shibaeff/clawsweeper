## Summary

Current main contains shipped fixes for the reported symptom cluster: false-negative gateway/status timeouts, network-interface discovery failures on Windows/WSL2-style hosts, and subagent runs being reported as timed out when they had actually finished. The issue’s later `status --deep` comment also already showed the gateway as reachable, which matches the current corrected behavior.
