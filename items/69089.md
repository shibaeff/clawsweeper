## Summary

Keep open. Current `main` does not implement the requested LaunchAgent refresh for unloaded gateway services, and this PR also bundles an unrelated commit-hash change that conflicts with current tested `resolveCommitHash` behavior.

## Evidence

- **Main still gates service-env refresh on loaded service:** `refreshGatewayServiceEnv = true` is still assigned only inside `if (loaded)`, so an unloaded gateway service does not take the refresh path on current `main`. That means the PR's stated bug is not already implemented on `main`.
