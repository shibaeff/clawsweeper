## Summary

Keep open. Current `main` does not implement this PR; it codifies the opposite packaging policy by documenting `node-llama-cpp` as an operator-installed local runtime and by failing release metadata if `node-llama-cpp` is added as a dependency. The branch also conflicts with current `main` safeguards that preserve the local-embedding mock seam and fallback typings, so this needs maintainer judgment rather than conservative auto-close.
