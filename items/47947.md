## Summary

Keep open. Current `main` still permits the malformed payload shape this PR targets: the WebSocket payload builder forwards `tool_choice` even when `tools` would be omitted, and the embedded-runner path still injects `parallel_tool_calls` without any final guard that strips tool-related fields when `tools` is absent or empty.
