## Summary

Current `main` already handles smaller-window switches conservatively: it clears stale cached context limits on model override, preflights prompt size against the active model budget before submission, attempts compaction/tool-result truncation recovery, and falls back to an explicit context-overflow reply instead of wedging the gateway.

## Evidence
