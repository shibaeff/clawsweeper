## Summary

Keep open. Current `main` does not contain the outbound message-dedupe feature from PR #43776, and the published PR patch still shows unresolved type, correctness, and test-isolation problems. There is no strong basis to close this PR as implemented, obsolete, incoherent, or out of scope.

## Evidence
