## Summary

Keep open. Current `main` still unconditionally renders `Compacted (before → after)` for manual `/compact`, so the reported UX bug is plausible, but this PR should not be auto-closed as implemented because its proposed `currentTokenCount` fix appears incorrect: `HandleCommandsParams.contextTokens` is the model context window, while the compactor contract expects caller-observed live prompt usage tokens.
