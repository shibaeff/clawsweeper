## Summary

Keep open. Current `main` does not implement a cron `script` payload in core, CLI, protocol, or Control UI, so this PR is neither already shipped nor obsolete. The provided review thread also records unresolved correctness issues in the proposed implementation around path resolution and delivery semantics, so there is no conservative close reason.
