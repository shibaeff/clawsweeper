## Summary

Keep open. Current `main` only partially addresses the stopword-noise problem: it adds a few chat-scaffolding words, but it does not implement the broader stopword/role-marker expansion or targeted coverage in this PR, and the linked issue discussion shows the dreaming-noise problem was still broader than this patch alone.

## Evidence

- **Current main only has a small English stopword set:** `LANGUAGE_STOP_WORDS.english` on current `main` contains 11 entries (`and`, `are`, `for`, `into`, `its`, `our`, `the`, `then`, `were`, `you`, `your`). The broader PR tokens such as `been`, `but`, `can`, `not`, `very`, `was`, plus role markers like `content`, `function`, `message`, `response`, and `role`, are not present there.
