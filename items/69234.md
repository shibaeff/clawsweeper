## Summary

Current main treats full compaction-strategy customization as plugin scope, not as another core openclaw.json prompt-template override. Core already covers the reporter's practical needs with language-preserving default instructions, extra custom compaction instructions, identifier-policy overrides, and a pluggable compaction-provider API for fully replacing built-in summarization.
