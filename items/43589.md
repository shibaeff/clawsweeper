## Summary

Keep open. Current `main` still uses glob-style chokidar watch targets for both skills and builtin memory, and the current tests explicitly lock in that behavior instead of covering the late-directory regression this PR targets. The PR discussion also shows the only concrete review findings were addressed in follow-up commit `7ed9f11`, so this still looks like actionable work rather than something safe to close.
