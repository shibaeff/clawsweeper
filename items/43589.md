## Summary

Keep open. Current `main` does not contain this PR’s watcher rewrite: the skills watcher still passes glob targets to `chokidar.watch()`, and the memory watcher still uses recursive glob targets for extra memory directories. The associated tests on `main` also still assert the old glob-based behavior rather than the proposed root-directory watching and path filtering.
