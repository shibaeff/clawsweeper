## Summary

Keep open. Current `main` still uses the suffix-overlap merge heuristic in the Gateway chat stream path, and the repeated-token loss case is still reproducible on that live segmented-text path. No equivalent fix or regression test is present on `main` or in the latest release snapshot inspected.
