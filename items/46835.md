## Summary

Keep open. Current `main` still handles chat session switching with a single header dropdown, not a multi-conversation left-sidebar list. The left sidebar is tab navigation only, and the separate Sessions page links into chat instead of surfacing all active channel conversations in the chat sidebar, so #46835 is not implemented on current `main`.
