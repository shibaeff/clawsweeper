## Summary

Issue #43058 is obsolete on current main: `agents.defaults.heartbeat` is still a strict schema with no `enabled` field, current docs do not document that key, the doctor flow explicitly strips unrecognized config keys, and the reporter confirmed on 2026-03-11 that `openclaw doctor --fix` removed the stray key and resolved the problem.
