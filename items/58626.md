## Summary

Keep open. Current `main` still treats a missing `plugins.load.paths` entry as a hard discovery error, and the only bundled-path doctor repair on `main` rewrites legacy `extensions/...` source-tree paths to the current `dist/extensions/...` layout under the same package root. It does not rewrite stale versioned package roots from older global npm/pnpm installs like the path in #58626. The special invalid-config reinstall escape hatch is also limited to bundled plugins that opt in; `matrix` opts in, but `acpx` does not.
