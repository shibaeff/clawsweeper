## Summary

Keep open. Current `main` still does not rewrite workspace-local skill paths to workspace-relative form before snapshot/prompt generation, and embedded sandbox runs reuse the host-built `skillsSnapshot.prompt`. That means the PR’s core sandbox-path bug is still plausibly real on `main`. One review concern from the PR history was addressed independently: current `main` already applies prompt char-budget checks to the compacted prompt-facing skill list, so the remaining gap is the missing workspace-relative compaction itself.
