## Summary

Current `main` already implements Feishu multi-account WebSocket startup and per-account status reporting. The key multi-account monitor logic and regression tests landed in commit `a6aa151653ab478e9aa23e95c6446c41adc4fce1` on April 24, 2026, after the reported `2026.3.28` release. That fix is present on `main` (`11ad1919edc7278f9d3e36c87c56bd3cb52c6e41`) but not in the latest tagged release `v2026.4.23`, so `fixedRelease` is unknown/not yet shipped.

## Evidence

- **Feishu monitor now starts all enabled accounts:** `monitorFeishuProvider` enumerates `listEnabledFeishuAccounts(cfg)` and launches `monitorSingleAccount(...)` for each configured Feishu account instead of only using a default account.
  - file: [extensions/feishu/src/monitor.ts:56](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/extensions/feishu/src/monitor.ts#L56)

  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Gateway startup is per-account:** Gateway channel startup resolves `plugin.config.listAccountIds(cfg)` and invokes `startAccount` once per `accountId`, so Feishu account runtimes are started individually.
  - file: [src/gateway/server-channels.ts:299](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/gateway/server-channels.ts#L299)

  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)
- **Channel status builds snapshots for every account:** `channels.status` collects `accountIds` from the plugin, builds an account snapshot for each one, and returns them in `payload.channelAccounts[channelId]`.
  - file: [src/gateway/server-methods/channels.ts:263](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/gateway/server-methods/channels.ts#L263)
  - command: `nl -ba src/gateway/server-methods/channels.ts | sed -n '258,335p'`
  - sha: [11ad1919edc7](https://github.com/openclaw/openclaw/commit/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41)

  - file: [src/commands/channels/status.ts:124](https://github.com/openclaw/openclaw/blob/11ad1919edc7278f9d3e36c87c56bd3cb52c6e41/src/commands/channels/status.ts#L124)

  - file: [extensions/feishu/src/monitor.startup.test.ts:55](https://github.com/openclaw/openclaw/blob/a6aa151653ab478e9aa23e95c6446c41adc4fce1/extensions/feishu/src/monitor.startup.test.ts#L55)
