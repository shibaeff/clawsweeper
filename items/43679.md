## Summary

Keep open. Current `main` still only suppresses empty/silent/heartbeat reply payloads and does not implement the proposed guard against leaked internal orchestration text, so this PR is not obsolete. The review thread also shows substantive iteration on false-positive and media-handling edge cases rather than evidence that the work was superseded elsewhere.
