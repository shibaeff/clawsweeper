## Summary

Current main already implements the requested skills-first behavior: the agent system prompt makes scanning `available_skills` mandatory before replying, the skills prompt injects installed skill locations into that catalog, the behavior is covered by tests, and the shipped `v2026.4.23` release already contains the same code.

## Evidence
