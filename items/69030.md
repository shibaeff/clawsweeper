## Summary

Current `main` still implements `openclaw infer model run` as an agent-runtime turn, not a raw model call. The CLI surface has no `--raw` flag, the implementation still resolves the default agent and routes through `agentCommand`/gateway `agent`, the docs explicitly describe this behavior, and the issue is only 6 days old with a concrete reproduction and proposed direction. Conservative cleanup should keep it open.
