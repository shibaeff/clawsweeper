## Summary

Keep open. Current `main` does not implement the PR's main unshipped behavior: Feishu `streaming` config is still boolean-only, docs still describe `true|false`, and there is no config path for per-session `throttleMs` or `title`. One part of the PR is partially obsolete because current `main` already has test coverage showing auto-mode markdown uses the streaming session, but that is not enough to close the PR as implemented.
