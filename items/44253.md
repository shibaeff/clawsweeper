## Summary

Keep open. Current `main` does not implement a per-agent `tools.selfDeny`/delegate-only mechanism, and the existing tool-policy pipeline still treats parent denies as final before subagent filtering.

## Evidence

- **Agent tool config has no self-only deny key:** `AgentToolsConfig` exposes `profile`, `allow`, `alsoAllow`, `deny`, `byProvider`, `elevated`, `exec`, `fs`, `loopDetection`, and `sandbox`, but no `selfDeny`, `delegateOnly`, or per-agent `subagents` tool override.
