## Summary

Current `main` does not implement this fix. Embedded sandboxed runs still prefer the prebuilt skill snapshot prompt verbatim, that snapshot prompt is built with host-compacted paths like `~/...`, and sandboxed read-path guards reject non-sandbox-root paths. The PR appears to address a real bug, but the discussion also identified unresolved review concerns, so this is not a conservative close candidate.
