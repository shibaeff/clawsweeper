## Summary

Keep open. Current `main` still does not implement the three requested config keys. The schema only allows `compaction.mode` as `default|safeguard`, `compaction.memoryFlush` only exposes `enabled`, `softThresholdTokens`, `forceFlushTranscriptBytes`, `prompt`, and `systemPrompt`, and `memorySearch.cache` only exposes `enabled` and `maxEntries`. There are adjacent mitigations on `main` (`forceFlushTranscriptBytes`, compaction providers, and content-hash-based embedding cache invalidation), but they are not the requested hard token threshold, priority-preserving compaction strategy, or cache TTL.
