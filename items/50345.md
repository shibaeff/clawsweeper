## Summary

Current main already carries the Anthropic thinking-block replay fix. The embedded runner now preserves the latest assistant thinking/redacted_thinking blocks during transcript sanitization for newer Claude models, and regression tests cover Anthropic replay with signed thinking blocks. The fix is present in release v2026.4.22.
