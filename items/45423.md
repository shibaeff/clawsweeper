## Summary

Current main no longer uses the legacy browser extension relay path that this issue reports. The relay path and `browser.relayBindHost` were removed, legacy `driver: "extension"` configs are auto-migrated to Chrome MCP `existing-session`, and the current docs explicitly say Docker/remote setups should use a raw CDP profile instead.
