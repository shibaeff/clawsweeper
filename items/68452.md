## Summary

Keep open: current `main` still only treats assistant-emitted `NO_REPLY` as a silent success. The embedded finalization path does not inspect `messagesSnapshot` for a latest successful tool-result `NO_REPLY`, so the reported isolated-cron `payloads=0` seam is not implemented away on `main`.

## Evidence

- **Payload construction is still assistant-only:** `buildEmbeddedRunPayloads` is built from `assistantTexts`, `toolMetas`, `lastAssistant`, and `lastToolError`; it does not accept or inspect `messagesSnapshot` or the latest successful tool result. That means a tool-result-only `NO_REPLY` can still leave `payloads` empty.
  - file: [src/agents/pi-embedded-runner/run/payloads.ts:121](https://github.com/openclaw/openclaw/blob/b84e57fca38deacfa9834a82f7e270b32562ae1b/src/agents/pi-embedded-runner/run/payloads.ts#L121)
  - command: `nl -ba src/agents/pi-embedded-runner/run/payloads.ts | sed -n '1,260p'`
  - sha: [b84e57fca38d](https://github.com/openclaw/openclaw/commit/b84e57fca38deacfa9834a82f7e270b32562ae1b)
- **Incomplete-turn suppression only trusts assistant `NO_REPLY`:** `resolveIncompleteTurnPayloadText` returns early only when `hasOnlySilentAssistantReply(assistantTexts)` is true. The helper's `IncompleteTurnAttempt` shape does not include `messagesSnapshot`, so a latest successful tool-result `NO_REPLY` cannot satisfy this guard.
  - file: [src/agents/pi-embedded-runner/run/incomplete-turn.ts:170](https://github.com/openclaw/openclaw/blob/b84e57fca38deacfa9834a82f7e270b32562ae1b/src/agents/pi-embedded-runner/run/incomplete-turn.ts#L170)

- **Main run path still errors on zero payloads after assistant/tool-media merge:** `runEmbeddedPiAgent` computes `payloadCount` from `buildEmbeddedRunPayloads(...)` plus `mergeAttemptToolMediaPayloads(...)`, then calls `resolveIncompleteTurnPayloadText(...)`, and on a truthy result logs `incomplete turn detected ... payloads=0` before returning an error payload. There is no intermediate fallback that synthesizes silent success from the latest successful tool result.
  - file: [src/agents/pi-embedded-runner/run.ts:1793](https://github.com/openclaw/openclaw/blob/b84e57fca38deacfa9834a82f7e270b32562ae1b/src/agents/pi-embedded-runner/run.ts#L1793)
