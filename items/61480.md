## Summary

Keep open. Current `main` still does not honor `HTTP_PROXY`/`HTTPS_PROXY` for ordinary `web_fetch` URLs; the code paths and shipped tests explicitly preserve strict DNS-pinned fetch behavior instead of opting into env-proxy mode.

## Evidence

- **web_fetch call site stays on strict guarded fetch:** `runWebFetch()` calls `fetchWithWebToolsNetworkGuard()` with URL, redirects, timeout, lookupFn, SSRF policy, and headers only. It does not pass `useEnvProxy` or a proxy dispatcher policy for normal `web_fetch` requests.
  - file: [src/agents/tools/web-fetch.ts:414](https://github.com/openclaw/openclaw/blob/7b5307acfc347a4e6d6647156600ee4b8d6eaa37/src/agents/tools/web-fetch.ts#L414)
  - command: `nl -ba src/agents/tools/web-fetch.ts | sed -n '389,430p'`
  - sha: [7b5307acfc34](https://github.com/openclaw/openclaw/commit/7b5307acfc347a4e6d6647156600ee4b8d6eaa37)
- **helper only enables env-proxy when explicitly requested:** `fetchWithWebToolsNetworkGuard()` uses `withTrustedEnvProxyGuardedFetchMode(...)` only when `useEnvProxy` is true; otherwise it wraps requests with `withStrictGuardedFetchMode(...)`.
  - file: [src/agents/tools/web-guarded-fetch.ts:45](https://github.com/openclaw/openclaw/blob/7b5307acfc347a4e6d6647156600ee4b8d6eaa37/src/agents/tools/web-guarded-fetch.ts#L45)
  - command: `nl -ba src/agents/tools/web-guarded-fetch.ts | sed -n '37,49p'`
  - sha: [7b5307acfc34](https://github.com/openclaw/openclaw/commit/7b5307acfc347a4e6d6647156600ee4b8d6eaa37)
- **SSRF guard only builds EnvHttpProxyAgent in trusted env-proxy mode:** `fetchWithSsrFGuard()` creates `EnvHttpProxyAgent` only when the guarded-fetch mode is `TRUSTED_ENV_PROXY` and proxy env is configured. In the default strict mode used by `web_fetch`, that branch is not taken.
  - file: [src/infra/net/fetch-guard.ts:357](https://github.com/openclaw/openclaw/blob/7b5307acfc347a4e6d6647156600ee4b8d6eaa37/src/infra/net/fetch-guard.ts#L357)

  - file: [src/agents/tools/web-tools.fetch.test.ts:256](https://github.com/openclaw/openclaw/blob/a9797214338ba31c52c796adbb75afb16e0684a9/src/agents/tools/web-tools.fetch.test.ts#L256)
