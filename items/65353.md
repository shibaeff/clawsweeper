## Summary

Keep open. Current main already has explicit `/new`/`/reset` startup-context loading for recent daily memory, so the reported amnesia symptom is at least partially mitigated, but I did not find equally strong evidence that mixed-install stale CLI paths or user-pinned automation entrypoints are now detected, warned on, or repaired well enough to conservatively close the issue.
