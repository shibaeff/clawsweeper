## Summary

Current `main` already skips exec script preflight for fully trusted gateway executions, including the `elevated="full"` path this PR targets, so #65419 is redundant.

## Evidence

- **`elevated=full` already collapses to full/off on main:** In current `main`, `elevatedMode === "full"` forces `security = "full"`, computes `bypassApprovals`, and then forces `ask = "off"`. That is the same trusted-session state the PR wanted to exempt from script preflight.
