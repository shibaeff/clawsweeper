## Summary

Keep open. Current `main` still has Gmail-only webhook/watch integration; the only shipped non-Gmail email support I found is the new bundled `skills/himalaya` skill for manual IMAP/SMTP operations, which is not the Gmail-style Gateway hook/webhook flow requested here.
