## Summary

Keep open. Current `main` supports parent transcript inheritance for Discord thread sessions and exposes manual pin actions, but it does not automatically fetch or inject parent-channel pinned messages into Discord thread session context, nor does it expose a dedicated `inheritPins`-style config.

## Evidence
