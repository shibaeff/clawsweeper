## Summary

Useful RFC, but current main already exposes the underlying agent-to-agent transport, delivery, and transcript/session primitives. The remaining `teleconference` work is an opinionated workflow/orchestration wrapper, and the repo vision explicitly pushes optional capabilities like this to plugins/skills via ClawHub rather than adding new core surface area.
