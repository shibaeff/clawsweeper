## Summary

Keep open. The reported silent gap still maps to a synchronous pre-ready plugin bootstrap path on current `main`, and Windows is still excluded from the native Jiti fast path that recent plugin-startup perf work relies on. Current docs and tooling add startup tracing/benchmarking, but that is observability evidence, not proof that the Windows regression is fixed.
