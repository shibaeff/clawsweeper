## Summary

Keep open. Current `main` and the latest tagged release still intentionally ship the Docker runtime on `node:24-bookworm` / `node:24-bookworm-slim`, and docs/tests codify that baseline. This issue is a recent, coherent Docker/runtime policy request, not something already implemented or stale enough to close.
