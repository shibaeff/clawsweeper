## Summary

Keep open. Current `main` does not include compaction summary text in `PluginHookAfterCompactionEvent`, so the requested hook-payload addition is not implemented. There is a workable fallback because the hook already exposes `sessionFile` and compaction summaries are persisted as `type: "compaction"` entries with a `summary` field, but that is not the same as adding `summaryText` directly to the hook event.
