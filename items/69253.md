## Summary

Current main already covers the PR’s goal through existing Gateway surfaces: `agent.identity.get` resolves the effective/default agent identity, and `agents.files.get` already exposes allowlisted bootstrap files including `IDENTITY.md` and `SOUL.md` through config-aware, workspace-safe reads. That makes `agent.identity.full` redundant as a core RPC.
