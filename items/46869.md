## Summary

Keep open. Current `main` still performs Scheduled Task restart inline (`/End` followed later by `/Run`) in `restartScheduledTask`, so this PR's detached handoff is not already implemented there. The detached helper exists only for the separate Windows self-restart path, and the PR's latest cleanup-before-handoff ordering still appears unsafe because the listener-cleanup fallback can target the running gateway PID before the detached relaunch script is spawned.
