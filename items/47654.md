## Summary

Keep open. Current `main` still rejects unknown config keys at load time, the gateway still fails closed on invalid config snapshots, and the docs still describe unknown legacy keys as validation failures until removed. The PR’s motivating Telegram `inlineButtons` example is outdated because that field is now in schema, but the broader core behavior requested here is not implemented on `main` and the review discussion surfaced real integration risks rather than evidence that the PR is obsolete.
