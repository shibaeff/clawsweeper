## Summary

Current main already routes interactive TUI chat runs through the fallback-aware reply runner, and GPT-5 terminal empty/plan-only/incomplete outcomes are now reclassified into model fallback so the session can advance to configured fallback models instead of stopping on the first surfaced failure.
