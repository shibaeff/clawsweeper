## Summary

Current main already gives heartbeat runs the low-context path the issue asked for: `isolatedSession: true` creates a fresh `:heartbeat` session with `forceNew: true`, `lightContext: true` limits bootstrap injection to `HEARTBEAT.md`, and the v2026.4.14 gateway/session fix stopped heartbeat-triggered session metadata from polluting later runs.

## Evidence
