## Summary

Current main only lets nodes declare command names, not per-command input/output schemas. The gateway protocol accepts `commands` as `string[]`, `node.describe` returns `commands?: string[]`, and `node.invoke` accepts untyped `params: unknown`, so the requested schema-declaration mechanism is not implemented on main.

## Evidence
