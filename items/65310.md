## Summary

The report is about an external IMA knowledge-base skill/integration, not an OpenClaw core surface on current `main`. Current Tencent support in this repo is limited to the TokenHub provider, the issue-specific API names/endpoints are absent from the tree, and project scope documents route new skills/plugins to ClawHub rather than core.
