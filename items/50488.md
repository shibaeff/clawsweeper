## Summary

Keep open. Current `main` has typing-delay behavior for unmentioned group chats, which partially addresses the UX flicker described in #50488, but it does not implement an optional pre-screen triage/classifier model. The existing path still relies on the main agent turn deciding to stay silent via `NO_REPLY`, so the cost-saving/core feature request is not implemented on `main`.
