## Summary

Keep open. Current `main` has partial scaffolding to merge tool-result media into cron payloads, but the embedded-run attempt result never populates `toolMediaUrls`/`toolAudioAsVoice`, so the cron path still lacks the TTS media handoff the issue describes.

## Evidence

- **Cron run would merge tool media if present:** `src/agents/pi-embedded-runner/run.ts` builds payloads, then calls `mergeAttemptToolMediaPayloads({ payloads, toolMediaUrls: attempt.toolMediaUrls, toolAudioAsVoice: attempt.toolAudioAsVoice })`. That is the only visible cron-side hook for tool-result media in the embedded runner.
  - file: [src/agents/pi-embedded-runner/run.ts:1642](https://github.com/openclaw/openclaw/blob/9153598e659daf5a52c479731f5b4ab8a5bf16b7/src/agents/pi-embedded-runner/run.ts#L1642)
  - command: `nl -ba src/agents/pi-embedded-runner/run.ts | sed -n '1638,1668p'`
  - sha: [9153598e659d](https://github.com/openclaw/openclaw/commit/9153598e659daf5a52c479731f5b4ab8a5bf16b7)
- **Attempt result never returns the needed media fields:** The embedded-run attempt return object includes `assistantTexts`, `toolMetas`, messaging-tool fields, and `successfulCronAdds`, but does not include `toolMediaUrls` or `toolAudioAsVoice`, even though those fields exist on the result type. Without those fields being populated, the merge hook above is a no-op.
  - file: [src/agents/pi-embedded-runner/run/attempt.ts:2776](https://github.com/openclaw/openclaw/blob/9153598e659daf5a52c479731f5b4ab8a5bf16b7/src/agents/pi-embedded-runner/run/attempt.ts#L2776)

- **Types show intended wiring is still incomplete:** `EmbeddedRunAttemptResult` declares optional `toolMediaUrls` and `toolAudioAsVoice`, which matches the intended fix shape, but repo-wide search in this checkout only found those names in the type, the merge helper, the merge call, and helper tests; no producer populates them in the run attempt path.
  - file: [src/agents/pi-embedded-runner/run/types.ts:91](https://github.com/openclaw/openclaw/blob/9153598e659daf5a52c479731f5b4ab8a5bf16b7/src/agents/pi-embedded-runner/run/types.ts#L91)
