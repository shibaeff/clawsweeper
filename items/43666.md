## Summary

Keep open. Current `main` does not implement per-embedded-run browser tab cleanup. Tabs are tracked at session scope and cleaned up on session lifecycle/reset paths, while `runEmbeddedAttempt` still has no browser-tab finalizer. Node-proxy browser opens also still bypass tracking entirely.

## Evidence

- **Embedded run cleanup has no browser-tab close step:** The `runEmbeddedAttempt` `finally` block only clears timers, unsubscribes, detaches reply backend, clears the active run, and removes the abort listener; there is no call to browser cleanup here.
  - file: [src/agents/pi-embedded-runner/run/attempt.ts:2596](https://github.com/openclaw/openclaw/blob/28fc03c386e14db502429bbe7e062cbe90439ca1/src/agents/pi-embedded-runner/run/attempt.ts#L2596)
  - command: `nl -ba src/agents/pi-embedded-runner/run/attempt.ts | sed -n '2588,2625p'`
  - sha: [28fc03c386e1](https://github.com/openclaw/openclaw/commit/28fc03c386e14db502429bbe7e062cbe90439ca1)
- **Browser tabs are tracked by session key, not run id:** Opened tabs are recorded with `sessionKey: opts?.agentSessionKey`, and explicit close removes them with the same session key. This is session-scoped tracking rather than per-run tracking.
  - file: [extensions/browser/src/browser-tool.ts:535](https://github.com/openclaw/openclaw/blob/28fc03c386e14db502429bbe7e062cbe90439ca1/extensions/browser/src/browser-tool.ts#L535)
  - command: `nl -ba extensions/browser/src/browser-tool.ts | sed -n '525,590p'`
  - sha: [28fc03c386e1](https://github.com/openclaw/openclaw/commit/28fc03c386e14db502429bbe7e062cbe90439ca1)
- **Cleanup API only closes tabs for session keys:** `closeTrackedBrowserTabsForSessions` pulls tracked tabs from `trackedTabsBySession` via `takeTrackedTabsForSessionKeys(sessionKeys)` and closes them. There is no run-level cleanup API.
  - file: [extensions/browser/src/browser/session-tab-registry.ts:114](https://github.com/openclaw/openclaw/blob/28fc03c386e14db502429bbe7e062cbe90439ca1/extensions/browser/src/browser/session-tab-registry.ts#L114)
  - command: `nl -ba extensions/browser/src/browser/session-tab-registry.ts | sed -n '114,179p'`
  - sha: [28fc03c386e1](https://github.com/openclaw/openclaw/commit/28fc03c386e14db502429bbe7e062cbe90439ca1)
- **Current cleanup callsites are lifecycle/reset paths, not normal embedded-run completion:** Browser cleanup is wrapped by `cleanupBrowserSessionsForLifecycleEnd`, then invoked from session reset cleanup, cron isolated-run cleanup, and subagent completion cleanup.
  - file: [src/browser-lifecycle-cleanup.ts:15](https://github.com/openclaw/openclaw/blob/28fc03c386e14db502429bbe7e062cbe90439ca1/src/browser-lifecycle-cleanup.ts#L15)
