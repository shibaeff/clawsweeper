## Summary

Keep open. Current `main` already has a Control UI exec-approval modal with countdown and one-click `allow-once` / `allow-always` / `deny`, but the command is still rendered as plain monospace text rather than syntax-highlighted, and the requested execution-context labeling is only generic metadata (`Host`, `Security`, `Ask`). That is not strong enough evidence to close this feature request as fully implemented.
