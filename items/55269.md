## Summary

Keep open. Current `main` does not expose a `commands.shellProfile` or equivalent exec-shell-profile setting, and the exec path still runs through the selected shell with `-c` while only merging login-shell `PATH`. The reporter’s linked PR #52154 is still open and contains the proposed implementation, so this request is neither implemented nor obsolete.
