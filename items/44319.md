## Summary

Keep open. Current `main` still builds the per-model extra-params lookup key with raw string concatenation, so provider-prefixed model IDs like `openrouter/auto` would still miss config under `agents.defaults.models`. The PR’s fix has not been implemented on `main` in the current codebase.

## Evidence
