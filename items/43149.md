## Summary

Keep open. Current `main` now covers the reporter's Cloudflare Markdown use case by automatically sending `Accept: text/markdown...` and handling `text/markdown` responses, but it does not implement the requested configurable `tools.web.fetch.defaultHeaders` option in the config/schema/docs.

## Evidence
