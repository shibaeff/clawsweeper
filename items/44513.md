## Summary

Keep open. Current `main` has a per-agent `agents.list[].reasoningDefault` override, but it does not implement the requested global default, and the fallback path still turns reasoning visibility `on` for reasoning-capable models when no session or per-agent override is set.

## Evidence

- **Maintainer context already distinguishes per-agent vs global:** The provided issue context includes a maintainer comment: `#51974` added per-agent reasoning defaults, but not a global default, and this issue was kept open as the follow-up.
  - command: `provided GitHub context`
- **Global agent defaults type has no reasoningDefault field:** `AgentDefaultsConfig` includes `thinkingDefault`, `verboseDefault`, and `elevatedDefault`, but no `reasoningDefault`, so `agents.defaults.reasoningDefault` is not part of the current typed config surface.
