## Summary

Current `main` still has only the implicit native-vs-bundle trust model. The PR's explicit `trustTier` type/loader/CLI/docs work has not landed, and the existing review comment identifies a real consistency issue that would need addressing if the PR is revived. Conservative cleanup should keep this PR open.
