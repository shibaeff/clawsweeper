## Summary

Current `main` already exposes `USER.md` and `MEMORY.md` in the Control UI, but only through the generic Agents `Files` tab. The requested dedicated memory surface is not implemented as described, and the backend whitelist does not include dated `memory/YYYY-MM-DD.md` files, so the issue is not strong enough to close as implemented.
