## Summary

Keep open. Current `main` and the latest shipped release still use a 3-minute resume watchdog cap, bundled CLI plugins inherit that shared default, and the runner still enforces it as a hard no-output kill. The PR addresses a real current-main behavior, although it likely needs test coverage updates before merge.
