## Summary

Current `main` already contains Feishu/Lark startup-preflight logic and regression coverage for restart-time startup ordering, and the same logic was already shipped in `v2026.4.23`. The report also cites an older mixed-version plugin/host stack that is below the current Feishu plugin compatibility floor.
