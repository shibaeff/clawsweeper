## Summary

Current `main` does not show a completed fix for automatic rename/migration between `openclawwechat` and `openclaw-weixin`. The repo now documents the official WeChat plugin as `@tencent-weixin/openclaw-weixin` / `openclaw-weixin`, but installer code still lacks a generic `replaces`/old-id cleanup path for install-time leftovers, so the issue should stay open.
