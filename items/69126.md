## Summary

Keep open. Current `main` has global `agents.defaults.compaction` settings, but it does not implement the requested `agents.defaults.sessionCompactionLimit` default or any `session_status` `compactionLimit` parameter. Auto-compaction is still described as token-threshold driven by the Pi runtime, so this feature request is not already satisfied on `main`.

## Evidence
