## Summary

Keep open. Current `main` and the latest release still hardcode a 120,000 ms gateway timeout for `openclaw infer model run --gateway`, which matches the reporter’s exact failure string and reproduction path. The command now defaults to local transport, but forcing `--gateway` still uses the capped timeout, so there is not strong evidence that the reported gateway-path bug was fixed or became obsolete.
