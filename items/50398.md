## Summary

Keep open. Current `main` still ships the bundled `coding-agent` skill with `bash ... background:true` guidance and an `openclaw system event` completion-notify pattern, while core docs now recommend `sessions_spawn`/`sessions_yield` for larger delegated tasks and describe push-based completion back to the requester.
