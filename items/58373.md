## Summary

Keep open. Current `main` does not contain this PR’s bootstrap logic: the `skip` path in `ensureOpenClawModelsJson` still returns immediately without creating a missing non-main `models.json`, while `agents.create` still does not create that file. The asymmetry with main-agent auth inheritance is still present, and the regression test described in the PR is not on `main`.
