## Summary

Keep open. Current `main` has partial neighboring cleanup for stale `plugins.allow`/`plugins.entries` refs and legacy bundled path rewrites, but it does not implement the PR’s full scope: there is no main-branch equivalent for pruning removed-plugin `plugins.installs`/`plugins.load.paths` or redundant bundled `discord` overrides, and `gateway status`/doctor still do not surface service version drift or explicit `~/.openclaw-tests/*` state-dir drift.
