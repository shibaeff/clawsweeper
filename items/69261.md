## Summary

Keep open. Current `main` still applies Ollama embedding auth without scoping it to the destination host, and the in-repo tests still assert the placeholder-token and local-host env fallback behavior this PR is trying to correct. The PR is therefore not obsolete, and its review thread also contains unresolved semantics feedback that needs maintainer judgment rather than cleanup-closing.
