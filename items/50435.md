## Summary

Keep open. Current `main` still reproduces the reported Feishu filename bug, so this is not implemented on `main`, but the PR’s proposed unconditional Latin-1->UTF-8 recovery is also not safe enough to close out as obsolete because it rewrites some valid non-ASCII filenames.
