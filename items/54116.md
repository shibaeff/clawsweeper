## Summary

Keep open. Current `main` does not implement a configurable Gemini web-search base URL: the Google plugin manifest exposes only `webSearch.apiKey` and `webSearch.model`, the runtime still hardcodes `DEFAULT_GOOGLE_API_BASE_URL`, and the user docs only document `apiKey`/`model`. The request remains coherent and in-scope for the bundled Google plugin, and the existing Google base-URL normalization helper supports the prior review feedback rather than weakening the PR.
