## Summary

Keep open. Current `main` still supports Slack assistant thread typing status via `assistant.threads.setStatus`, but there is no implementation for `assistant.threads.setSuggestedPrompts`, no `assistantPrompts` config surface, and no Slack message-tool action for setting suggested prompts. The nearest existing feature is Slack interactive replies via Block Kit directives, which is a different mechanism.
