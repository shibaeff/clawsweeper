## Summary

Keep open. Current `main` still has the exact path mismatch the PR describes: runtime looks for bundled hooks under `dist/hooks/bundled`, while the build script still copies `HOOK.md` files to `dist/bundled`. Because the published package ships `dist/` and not `src/`, the npm/global-install path remains plausibly broken rather than obsolete.
