## Summary

Keep open. Current `main` at `9b7f6250f43bbc1816cdaf26e8c07116dfa4975f` still does not implement the PR’s read-image fallback from `tool_execution_start` args, and the current tests still encode the missing fallback as expected behavior.

## Evidence

  - file: [src/agents/pi-embedded-subscribe.tools.ts:332](https://github.com/openclaw/openclaw/blob/9b7f6250f43bbc1816cdaf26e8c07116dfa4975f/src/agents/pi-embedded-subscribe.tools.ts#L332)
