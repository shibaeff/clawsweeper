## Summary

Keep open. Current `main` does not implement this PR’s queue diagnostics feature: the Gateway surface still exposes `diagnostics.stability` and diagnostics export, docs still tell users to inspect queue depth via verbose logs, and the command queue only exposes count-based helpers rather than a snapshot API with lane/session ages.
