## Summary

Current `main` intentionally treats MiniMax Anthropic chat models as text-only, routes image understanding through `MiniMax-VL-01`, and offloads attachments for text-only runs instead of silently dropping them. MiniMax’s current official Anthropic-compatible API docs also say image input is not supported, so this PR would advertise an upstream capability that is not available.
