## Summary

Keep open. Current main has partial progress on this field report for the managed Playwright path: selector-targeted actions and batched `/act` dispatch are covered by the control-route tests, and the browser tool description now points users to stable `refs="aria"` snapshots. But the issue as written is not implemented end-to-end on the browser tool surface: the tool schema still does not expose `batch`, the action contract still has no `inspectElement`/DevTools action and no ref-based wait, form fills still do not verify that values stuck, and snapshots still do not report CAPTCHA detection.
