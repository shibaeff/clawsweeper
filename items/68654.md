## Summary

Keep open. Current `main` does not implement this PR’s lazy plugin CLI optimization or its config-snapshot reuse, and the existing tests still encode the pre-PR behavior. The PR discussion also surfaced a real regression risk and a follow-up fix on the PR branch, so this is not stale or incoherent cleanup material.

## Evidence
