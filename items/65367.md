## Summary

Keep open. Current `main` fixes the old heartbeat-coupled managed-cron path, but the remaining reported gap is still real: dreaming still resolves all configured agent workspaces, processes them serially, and exposes only global dreaming config rather than per-agent/workspace scoping.

## Evidence
