## Summary

Keep open. Current `main` does not include a bundled Requesty provider, and the repo gives mixed but non-decisive scope guidance: OpenClaw supports external provider plugins via ClawHub/npm, but it also explicitly prioritizes support for major model providers and already ships comparable bundled proxy providers like OpenRouter. That makes this PR a maintainer product/scope decision, not a conservative auto-close.
