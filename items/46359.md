## Summary

Keep open. Current `main` already replaced Ollama discovery and stream with stronger guarded-fetch + pinned-DNS SSRF handling, but the vLLM/SGLang discovery path named in this PR still uses a raw `fetch()` with no SSRF guard. The submitted PR is also stale against current architecture and its review feedback identified functional regressions for default local endpoints, so there is not strong evidence to close it as implemented or obsolete.
