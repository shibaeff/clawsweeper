## Summary

Current `main` does not implement this fix. `setupSkills` still shows the Homebrew recommendation on every non-Windows platform, so the FreeBSD/OpenBSD bug described by the PR remains plausible and unshipped. The PR should stay open for maintainer judgment rather than be auto-closed.
