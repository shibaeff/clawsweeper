## Summary

Current main does not add `systemPromptAppend` as a core config key, but the requested behavior already fits the documented plugin seam: plugins can override model selection, append/prepend system-prompt context, and inspect the resolved model id/provider before the final prompt is sent. OpenClaw vision and docs explicitly steer optional prompt-behavior customization toward plugins/ClawHub rather than new core knobs.
