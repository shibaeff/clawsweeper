## Summary

Current `main` still defaults QMD collection bootstrapping to `--glob`, and the existing fallback only switches to `--mask` after an explicit unsupported-option error. That means the silent-ignore regression described in #65480 is not implemented away on `main`, and this PR still appears to address a live gap with targeted test coverage.
