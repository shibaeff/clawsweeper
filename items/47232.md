## Summary

Keep open. Current `main` does not implement the PR’s requested Edge TTS behavior: the Microsoft plugin still does a single `ttsPromise` call and throws immediately on a zero-byte file, with no empty/whitespace guard and no retry. The code has also moved from the PR’s old target path into the bundled Microsoft plugin, so the PR is stale in shape but not obsolete in substance.
