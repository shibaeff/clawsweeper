## Summary

Keep open. Current `main` has solid per-agent inspection/editing in Control UI, so parts of the proposal are already shipped, but the core ask remains unmet: there is still no cross-agent/team relationship view for delegation, shared-workspace dataflow, or task sequencing. The issue is recent, detailed, and actionable, so there is no conservative close reason.

## Evidence

## Risks / Open Questions

- Some proposed subfeatures are already implemented on `main`, so if this stays open it should likely be reframed around the remaining missing capability: cross-agent relationship visualization/debugging, not generic per-agent editing.
- A full solution likely needs either new gateway/protocol data for team edges or explicit filesystem/project discovery logic for `allowAgents`, shared-workspace symlinks, and `TASK.json`; current UI RPCs do not expose that shape yet.
