## Summary

Keep open. Current `main` does not contain this docs addition, and the PR still appears to have a material correctness problem: its watchdog gate uses `openclaw health --json`, but current `main` defines that command as a gateway-reachability check with channel issues reported but non-fatal, so it does not match the PR’s stated "process is alive but unhealthy" recovery goal.
