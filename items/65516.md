## Summary

Current `main` no longer matches the reported `bundle-a2ui.sh` failure, and the repo now documents `pnpm build` as a source-checkout workflow rather than something supported inside a packaged global install.

## Evidence

- **Current build script no longer references the missing shell wrapper:** On current `main`, the `canvas:a2ui:bundle` script runs `node scripts/bundle-a2ui.mjs`, so the reported `bundle-a2ui.sh not found` failure does not match present code.

  - file: [package.json:23](https://github.com/openclaw/openclaw/blob/1bdf5307d99d9c1edcbe099d7c2fc3947a9b0a19/package.json#L23)
