## Summary

Keep open. Current `main` and the latest release still interpolate `embeddingCacheTable` and `ftsTable` directly into DDL in `memory-schema.ts`, so the reported concern is not already fixed. The PR discussion also surfaces unresolved correctness risk in the proposed patch (`quoteTable` breaks pre-quoted identifiers) plus a remaining unconstrained `definition` fragment and no targeted tests for this module, so there is not strong evidence to close it as implemented, obsolete, or incoherent.
