## Summary

Issue #56172 requests WeChat recall/revoke support, but current main treats WeChat as an external channel plugin and keeps channel-specific message actions in the plugin boundary rather than OpenClaw core. Core already exposes generic `messageId` and `unsend` seams; the missing work is in `@tencent-weixin/openclaw-weixin`, not this repository.

## Evidence

- **WeChat is explicitly external to this repo:** The WeChat docs say OpenClaw connects through Tencent's external `@tencent-weixin/openclaw-weixin` plugin, that the WeChat code does not live in the OpenClaw core repo, and that Tencent iLink API calls/account monitoring are owned by the external plugin.
