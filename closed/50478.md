## Summary

Current `main` intentionally scopes `sessions_list` to visible sessions instead of a global runtime-wide list, and the contract is now explicit in code and docs: default visibility is `tree`, off-tree sessions are filtered, and the tool/docs describe the result as "visible sessions" with documented `self|tree|agent|all` scope levels.

## Evidence
