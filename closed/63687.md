## Summary

Current `main` already implements the requested Z.AI endpoint flexibility: the bundled Z.AI provider now supports endpoint auto-detection, explicit `zai-coding-cn` / `zai-cn` onboarding choices that use `open.bigmodel.cn`, and writes the selected endpoint into `models.providers.zai.baseUrl` in config.
