## Summary

Keep open. Current main already accepts `768P` for `video_generate`, so the video-validation half looks obsolete, but the MiniMax music path is still coded around `music-2.5+` defaults and request shaping that matches the remaining failure reports in the issue discussion. There is not strong evidence that the overall item is fixed on `main`.
