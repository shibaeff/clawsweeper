## Summary

Current `main` no longer uses the flat edit-parameter contract this PR targets. Edit execution now requires canonical `path` plus `edits[]`, and main has regression coverage explicitly rejecting the legacy alias payloads from this PR.

## Evidence

- **Current edit contract requires `path` + `edits[]`:** `REQUIRED_PARAM_GROUPS.edit` now validates only canonical `path` and `edits`, with `edits` required to be a non-empty array of `{ oldText, newText }` objects.
