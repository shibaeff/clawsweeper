## Summary

Current main already implements Feishu DM/group support with the Feishu doc/wiki tools registered on the channel’s full entrypoint, and that implementation was already shipped in v2026.4.22. The reported 2026.3.13 missing-tool gap is obsolete against current main.

## Evidence

- **Feishu full entry now registers doc/wiki tools:** The bundled Feishu channel entry calls `registerFeishuDocTools` and `registerFeishuWikiTools` from `registerFull`, which is the full-mode registration path for channel plugins.
  - file: [extensions/feishu/index.ts:73](https://github.com/openclaw/openclaw/blob/6944d7025d9c0ba4891cbe646ab5dab039d504ca/extensions/feishu/index.ts#L73)
  - sha: [6944d7025d9c](https://github.com/openclaw/openclaw/commit/6944d7025d9c0ba4891cbe646ab5dab039d504ca)
- **Doc tool is registered for Feishu channel context:** `registerFeishuDocTools` registers `feishu_doc`; the tool factory reads `ctx.agentAccountId` and Feishu requester context, showing it is intended to run inside Feishu channel sessions rather than only a local console path.
  - file: [extensions/feishu/src/docx.ts:1420](https://github.com/openclaw/openclaw/blob/6944d7025d9c0ba4891cbe646ab5dab039d504ca/extensions/feishu/src/docx.ts#L1420)
  - sha: [6944d7025d9c](https://github.com/openclaw/openclaw/commit/6944d7025d9c0ba4891cbe646ab5dab039d504ca)

  - file: [extensions/feishu/src/wiki.ts:174](https://github.com/openclaw/openclaw/blob/6944d7025d9c0ba4891cbe646ab5dab039d504ca/extensions/feishu/src/wiki.ts#L174)
