## Summary

Issue #63172 reports a real WeChat plugin problem, but the evidence on current `main` shows it belongs to the external `@tencent-weixin/openclaw-weixin` plugin rather than OpenClaw core. Current docs explicitly say the WeChat runtime, context tokens, and account monitoring are owned outside this repo, and there is no in-repo WeChat source to patch here.

## Evidence

- **WeChat docs mark this as an external plugin:** The WeChat docs say OpenClaw connects through Tencent's external `@tencent-weixin/openclaw-weixin` plugin, that the WeChat code does not live in the OpenClaw core repo, and that context tokens and account monitoring are owned by the external plugin.
