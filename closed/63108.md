## Summary

Current `main` no longer resolves bundled channel secret metadata by importing broad channel runtime surfaces. The secret target registry now loads lightweight `secret-contract-api.js` sidecars, which directly addresses the circular-init path described in #63108 and was shipped in `v2026.4.8` with follow-up hardening in `v2026.4.12`.

## Evidence
