## Summary

Current `main` already has the Config Form/Raw mode toggle wired and covered by browser tests, and the same implementation is present in the latest release `v2026.4.23`. The reported `content.js:1` SyntaxError does not map to any Control UI asset in this repo, so the old regression described in #63704 is not reflected in supported current builds.

## Evidence
