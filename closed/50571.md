## Summary

Current `main` closes the reported misrouting path: spawned subagent sessions now inherit delivery context from the triggering request, agent delivery resolution prefers the current turn source over drifting session `lastChannel` state, and subagent runs now fail closed unless the `message` tool gets an explicit target.

## Evidence

- **Subagent sessions inherit trigger delivery context:** The gateway `agent` handler now seeds a missing session `deliveryContext` from the request `channel`/`to`/`accountId`/`threadId`, with an inline comment describing the prior wrong-channel drift failure mode for freshly spawned subagents.
