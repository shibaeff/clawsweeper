## Summary

Current `main` no longer matches the reported v2026.4.2 LINE startup regression: the LINE runtime sidecar and manifest/package metadata are part of the bundled plugin pack surface, the facade loader now falls back through the manifest registry when direct bundled metadata lookup misses, and there is explicit regression coverage for resolving LINE `runtime-api.js` from a global plugin-style root.
