## Close Comment

Closing this as implemented after Codex review.

Current `main` already implements this behavior through a shared env-proxy bootstrap, with broader coverage and tests than this PR’s one-file change. The PR is redundant against `main` as checked at `7ac35b4f69009490180a35bf404638d4280a4865`.

What I checked:

- **CLI now bootstraps the shared env-proxy dispatcher before command routing:** `runCli()` calls `ensureGlobalUndiciEnvProxyDispatcher()` immediately after env normalization and before `tryRouteCli()`, which is the same placement this PR wanted for outbound CLI traffic. ([`src/cli/run-main.ts:187`](https://github.com/openclaw/openclaw/blob/7ac35b4f6900/src/cli/run-main.ts#L187), [7ac35b4f6900](https://github.com/openclaw/openclaw/commit/7ac35b4f6900))
- **Shared helper installs `EnvHttpProxyAgent` with safe fallback:** `ensureGlobalUndiciEnvProxyDispatcher()` checks for configured HTTP(S) proxy env, installs `EnvHttpProxyAgent`, and catches failures instead of crashing the CLI. That is a stronger centralized implementation than the PR’s inline block. ([`src/infra/net/undici-global-dispatcher.ts:91`](https://github.com/openclaw/openclaw/blob/7ac35b4f6900/src/infra/net/undici-global-dispatcher.ts#L91), [7ac35b4f6900](https://github.com/openclaw/openclaw/commit/7ac35b4f6900))
- **Regression coverage exists on main:** Dedicated tests cover installing the env proxy dispatcher, idempotence, and preserving unsupported custom dispatcher types. ([`src/infra/net/undici-global-dispatcher.test.ts:199`](https://github.com/openclaw/openclaw/blob/7ac35b4f6900/src/infra/net/undici-global-dispatcher.test.ts#L199), [7ac35b4f6900](https://github.com/openclaw/openclaw/commit/7ac35b4f6900))
- **Main applies the same bootstrap in other outbound paths too:** Current main also initializes the shared env-proxy dispatcher in other outbound HTTP entry points, including embedded-runner HTTP setup and Codex OAuth login, so the implementation is broader than this PR’s CLI-only patch. ([`src/agents/pi-embedded-runner/run/attempt-http-runtime.ts:10`](https://github.com/openclaw/openclaw/blob/7ac35b4f6900/src/agents/pi-embedded-runner/run/attempt-http-runtime.ts#L10), [7ac35b4f6900](https://github.com/openclaw/openclaw/commit/7ac35b4f6900))
