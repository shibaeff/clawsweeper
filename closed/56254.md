## Summary

Current main and the shipped v2026.4.22 already address the reported handshake timeout: the gateway pre-auth handshake default is 10s instead of 3s, `OPENCLAW_HANDSHAKE_TIMEOUT_MS` is supported outside VITEST, and current CLI startup policy does not preload plugins for `cron` commands.
