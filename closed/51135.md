## Summary

The request is a specialized coding-assistant package, and current main explicitly directs optional capability, new skills, and ecosystem-specific orchestration to ClawHub/community plugins instead of OpenClaw core. Main already provides one-command install paths for ClawHub plugins/skills and supports coding-focused bundles/plugins, so this belongs in the plugin ecosystem rather than as a core roadmap item.
