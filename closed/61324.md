## Summary

Current `main` already implements the reported fix: ACP delivery now asks channel plugins whether delivered block text is user-visible, Discord registers that visibility hook for non-empty block replies, and the final fallback only runs when no visible text was already delivered. The released `v2026.4.23` tree contains the same implementation.

## Evidence
