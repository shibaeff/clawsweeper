## Summary

Current `main` already implements this request: `gateway.channelStaleEventThresholdMinutes` is part of the public config, is validated and documented, and is wired into the channel health monitor as the stale-socket threshold. The released tag `v2026.4.22` also contains the same config surface, so the feature is already shipped.
