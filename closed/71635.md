## Summary

Keep open. Current `main` still misparses LM Studio model ids with `@iq*` quant suffixes as auth profiles, and this PR directly addresses that gap. The review comment about bare `i` overmatching was already incorporated into the PR's updated regex, so the remaining evidence supports maintainer review/merge rather than cleanup closure.

## Evidence
