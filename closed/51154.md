## Summary

Current `main` implements both sides of the report: it now breaks context-growth/tool-loop failure paths with explicit overflow handling and user-visible fallback text, and it has a supported gateway session-reset path that rotates the session in store/runtime instead of relying on deleting transcript files.

## Evidence

The guard is installed for embedded runs in `run/attempt.ts`.
