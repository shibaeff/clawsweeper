## Summary

Current `main` already covers this PR's intent. The compaction safeguard now uses the subsystem logger for the missing-model path and a dedicated `resolveModelAuth()` path that logs credential-resolution failures before cancelling compaction, so the silent-cancel logging gap described here is already implemented on `main` via commit `8cc38c1b86f41803447f4c7d962aa9465d5c8ae8`.
