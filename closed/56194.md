## Summary

Current main already preserves self-declared operator scopes for the direct-local `gateway-client`/`backend` shared-auth path described in the report, with explicit compat tests for both token and password auth. Generic device-less shared-auth operator connects still intentionally clear scopes, so the reported blanket regression is not present on current `main`.
