## Summary

Current main no longer has the reported crash path during config loading. `applyContextPruningDefaults` delegates to provider defaults, and the Anthropic defaults code now uses a local model-ref parser instead of `parseModelRef`/`ANTHROPIC_MODEL_ALIASES`. Current tests cover Anthropic `agents.defaults.model.primary` and `contextPruning` defaulting behavior. The released `v2026.4.22` tag contains the same runtime implementation.
