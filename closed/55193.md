## Close Comment

Closing from a Codex review because this regression is already implemented on `main` and shipped in `v2026.4.22`. The bundled plugin resolver now prefers the running CLI package root before the current working directory in `src/plugins/bundled-dir.ts:141`, and there is a targeted regression test for `prefers the running CLI package root over an unrelated cwd checkout` in `src/plugins/bundled-dir.test.ts:299`. If you still see this with a current install, please open a fresh issue with `openclaw --version`, install method, cwd, and any `OPENCLAW_BUNDLED_PLUGINS_DIR` override details.
