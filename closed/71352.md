## Summary

Keep open. Current `main` still drops `error.message` and `error.stack` from persisted stability bundles, so the underlying bug is not implemented on `main`. At the same time, current tests and docs define these bundles as payload-free and shareable, so this PR changes an explicit privacy contract and needs maintainer review rather than conservative auto-close.
