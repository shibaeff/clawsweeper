## Summary

Current `main` already caps the skill-list prompt surface and falls back to a compact format that omits descriptions when the full skills block would exceed budget, which addresses the reported prompt-bloat problem. It also truncates user-invocable skill command descriptions and documents the prompt budget and best practice to keep descriptions short.
