## Summary

Current `main` already preserves recoverable pre-compaction transcript state via persisted compaction checkpoints, with list/get/branch/restore flows and regression coverage. That shipped alternative covers the core request behind this PR, so the `.compaction.<timestamp>` archive approach here is redundant on current OpenClaw.
