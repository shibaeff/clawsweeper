## Summary

Current main already has public tool lifecycle hooks for OpenClaw agent sessions. `before_tool_call` can inspect and rewrite params, block execution, or require approval; `after_tool_call` receives params plus result/error metadata for logging or audit plugins. The Pi agent runtime wires these hooks around real tool execution, and the changelog shows the capability shipped by 2026.2.9, with `before_tool_call` first wired in 2026.1.31.
