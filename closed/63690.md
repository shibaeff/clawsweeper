## Summary

Current `main` already refreshes assistant identity and avatar state on session switches, and the current chat renderer now uses that refreshed identity/avatar in the welcome state and transcript groups. The reported Control UI behavior is implemented on `main` at commit `86f8c826e20a01d1bba67c0ba6e73182fbf0455f`.
