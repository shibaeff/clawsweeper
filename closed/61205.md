## Summary

Close as `clawhub`. Current main already has the underlying OpenAI Codex OAuth + device-code auth paths, plus plugin/native Telegram command surfaces. The remaining ask is a Telegram-specific reconnect UX wrapper, and `VISION.md` explicitly keeps auth/setup terminal-first while pushing optional capability into plugins/ClawHub rather than core.
