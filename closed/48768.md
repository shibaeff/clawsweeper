## Summary

Current `main` already contains the Control UI and model-selection fixes for provider-prefix drift in session model switches. The dashboard model picker now builds and sends provider-qualified `provider/model` values, preserves already-qualified refs from server state, and the allowlist resolver has regression coverage for preventing fallback to the current session provider. The changelog records these fixes as shipped in `v2026.4.2`, so this 2026.3.13 regression is obsolete on supported releases.
