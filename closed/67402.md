## Summary

Current `main` already contains targeted fixes for restart/system-event routing and heartbeat relay behavior that match this report: restart sentinel notices stay session-scoped with preserved delivery context, isolated wake runs avoid stale turn-source routing, and exec/cron heartbeat prompts switch to internal-only handling when there is no user-visible delivery target.
