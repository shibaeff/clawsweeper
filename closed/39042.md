## Summary

Current `main` already handles the false-positive duplicate case more conservatively than this PR: it silently deduplicates only when both candidates resolve to the same physical plugin directory, preserves origin precedence in place, and keeps warnings for true duplicates from different directories. That makes PR #39042 obsolete.
