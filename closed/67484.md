## Summary

Close as redundant. Current `main` already strips the `ollama/` prefix inside `buildOllamaChatRequest`, keeps the stream call site passing raw `model.id`, and has a regression test for the exact prefixed-model case. The same implementation is present in released `v2026.4.23`, so the PR's call-site-only patch is already superseded.
