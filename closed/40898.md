## Summary

Current `main` has already replaced the old subagent completion path with a newer announce pipeline that preserves parent-agent mediation. The specific 2-line default flip in this PR is no longer the right fix: active requesters are steered internally, dormant requesters still route completion through the parent agent session, and the e2e/tests explicitly assert that raw `✅ Subagent ...` output is not sent through.

## Evidence
