## Summary

The shipped v2026.4.23 code already gives subagent runs session-scoped bundle MCP tool materialization, keeps bundle-MCP tools enabled in normal coding/messaging profiles, and preserves subagent delivery context needed for Discord-targeted follow-up routing.
