## Summary

Current `main` already covers the reported failure mode: heartbeat runs now skip busy session lanes instead of interrupting active requests, and silent no-response timeouts now surface an explicit error. The relevant implementation and tests are on `main` in commit `14934f0b7c874e6af4ed93a1c307a17d222ef739`, which is newer than the latest tagged release `v2026.4.23` and appears unreleased so far.
