## Summary

Current `main` already contains a shipped fix for the reported later-turn `tool ... not found` skill/session drift. OpenClaw 2026.4.15 explicitly shipped session skills-snapshot invalidation plus an always-on unknown-tool guard for this failure mode, and current code/tests keep the session skill snapshot refreshed and tool-rich follow-up prompts stable.
