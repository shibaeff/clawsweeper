## Summary

Current main already provides a plugin SDK seam for sandbox backends and uses that seam for the managed OpenShell sandbox. This PR adds OpenSandbox as core-specific backend/config/env plumbing instead of using the plugin path, so it is out of scope for OpenClaw core and should be pursued as a plugin/backend package.
