## Summary

Current `main` no longer uses a static Ollama catalog entry for `deepseek-v3.2`; hosted Ollama models are discovered live from `/api/tags`, model definitions no longer set `supportsTools: false`, and tool support defaults to enabled unless a model explicitly opts out.

## Evidence
