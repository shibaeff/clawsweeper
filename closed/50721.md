## Summary

Current `main` already contains the boot-md startup-path fixes that match this report: boot runs per configured agent scope with per-agent session-key routing, the BOOT prompt still directs outbound sends through the message tool, and outbound channel resolution now preserves pinned channel adapters across post-startup registry replacement with regression tests covering that path.

## Evidence
