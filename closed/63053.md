## Summary

Current `main` already supports assistant-originated outbound media replies: `image_generate` auto-attaches outputs, assistant replies can include `MEDIA:<path-or-url>`, and the shared `message` tool accepts `media`/`path`/`filePath` for attachment sends. The reply pipeline also preserves media-only tool results, with Telegram/native delivery covered by tests.
