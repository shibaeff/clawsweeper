## Summary

Issue #50823 is a coherent idea, but current main positions this kind of game-like orchestration dashboard as plugin territory, not OpenClaw core. The repo vision explicitly keeps core lean, pushes optional capability to plugins/ClawHub, and excludes heavy orchestration layers as default architecture.

## Evidence
