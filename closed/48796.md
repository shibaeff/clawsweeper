## Summary

Current `main` already addresses the light-terminal readability problem in the TUI by keeping assistant body text on the terminal default foreground, adding a dedicated light palette, auto-detecting light backgrounds via `COLORFGBG`, and allowing an explicit override with `OPENCLAW_THEME=light|dark`.

## Evidence
