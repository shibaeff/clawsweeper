## Summary

Issue #55671 is implemented on current main: Gateway now persists assistant-generated images as managed media, records them in transcript-backed image blocks, and Control UI renders those image blocks through authenticated media fetches. The changelog and Control UI docs explicitly describe this fix, but it does not appear to be in the latest tagged release yet.
