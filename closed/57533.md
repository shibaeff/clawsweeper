## Summary

The proposal asks for a native core orchestration/safety framework, but current project direction explicitly keeps optional orchestration layers out of core and routes that kind of policy/automation into plugins/hooks/ClawHub. Main already provides the low-level primitives this proposal depends on: inter-session provenance, cross-agent/session gating, and synchronous hook points that can block or rewrite message persistence.
