## Summary

Current main already self-heals stale auto-selected session provider/model overrides and ships regression coverage for the OpenRouter-style fallback case described in #56206.

## Evidence

- **Auto override self-heal on reply path:** The reply model-selection path detects a direct auto session override, rewrites the session entry back to the configured default selection, persists that change, and resets the in-memory provider/model so the current turn retries the configured primary instead of the stale fallback.
