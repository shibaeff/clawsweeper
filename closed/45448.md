## Summary

Current `main` already covers this behavior through the shared plugin/hook update path: deprecated `hooks update` delegates to `runPluginUpdateCommand`, that command exits cleanly on `--all` when nothing is tracked, and the same code is present in released `v2026.4.22` with regression coverage.
