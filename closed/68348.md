## Summary

Current `main` already implements the underlying status/doctor loading optimization, and the latest release `v2026.4.23` ships it. The PR’s proposed `tryNative` Jiti tweak is obsolete because doctor contract JS modules now bypass Jiti entirely via native `require`, with tests covering both the native fast path and the Windows fallback.
