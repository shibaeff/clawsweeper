## Summary

Current main no longer matches the reported Slack socket-mode crash path: OpenClaw disables Slack SDK auto-reconnect, owns Slack reconnect handling with bounded backoff, and gateway channel failures are caught and restarted instead of surfacing as unhandled rejections.
