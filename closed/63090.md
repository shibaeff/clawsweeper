## Summary

PR #63090 adds a new optional memory skill under `workspace/skills`, but current OpenClaw policy and docs route new skills and optional memory capability to ClawHub/plugins rather than the main repo. Main already exposes memory through plugin-based surfaces, and the PR discussion still shows unresolved correctness issues, so this is better continued as an external skill/plugin.
