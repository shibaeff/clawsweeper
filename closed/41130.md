## Summary

Current main already implements native WebChat streaming over the Gateway WebSocket. The server emits incremental `chat` delta events, the WebChat UI consumes those deltas into `chatStream`, and the UI renders a live streaming bubble. The issue’s core premise no longer matches `main`.
