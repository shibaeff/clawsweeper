## Summary

Current `main` already addresses the reported dropped-tool-arguments/session-replay failure on the active embedded runner path. The live stream now repairs malformed OpenAI-compatible tool-call arguments before execution, and replayed session history sanitization drops malformed persisted tool calls on resume/restart.
