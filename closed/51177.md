## Summary

Current `main` already hides the raw full provider catalog from Control UI by serving the runtime-allowed model catalog through `models.list`, which the UI consumes directly. The new behavior is covered by docs and tests, but it does not appear to be in the latest tagged release yet.

## Evidence
