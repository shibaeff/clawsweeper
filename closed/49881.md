## Summary

Current `main` already exposes the two helpers on supported public SDK subpaths, so the PR's stated problem (`not reachable from any public surface`) no longer matches the codebase. `plugin-sdk/feishu` is now documented as a bundled-maintenance seam, not the recommended third-party import path.
