## Summary

Current `main` already loads workspace markdown bootstrap files for terminal/CLI runs, injects them into `Project Context`, and ships docs/tests that cover that behavior; the report does not describe a gap that still exists in the checked-out code.

## Evidence

  - file: [src/agents/workspace.ts:477](https://github.com/openclaw/openclaw/blob/608c08fc54da2f86338366f8eeafd7e2a02713aa/src/agents/workspace.ts#L477)
