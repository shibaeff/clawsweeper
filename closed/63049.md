## Summary

Current `main` already implements the Feishu `/stop` control-lane behavior this PR is trying to add, and the shipped `v2026.4.23` release already contains that implementation. The later PR discussion narrowed scope to abort-only routing via `isAbortRequestText`, which matches current `main` rather than the broader original PR description.
