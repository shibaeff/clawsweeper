## Summary

Current main already has native heartbeat scheduling and config. The requested capability exists under `agents.defaults.heartbeat` / `agents.list[].heartbeat`, with `every` as the interval key and `prompt` as the heartbeat body, and the latest release tree also contains that implementation.

## Evidence
