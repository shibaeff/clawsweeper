## Summary

Current `main` already contains the Control UI and session-model handling needed for this bug class, and the changelog says it shipped in `2026.4.2`. The chat header model picker now uses provider-qualified option values and sends the selected value unchanged in `sessions.patch`, while session merging also clears stale `modelProvider` carry-over when only `model` is patched.
