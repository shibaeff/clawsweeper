## Summary

Current `main` already addresses the reported Gemini CLI OAuth failure path: it detects personal Gemini OAuth and skips `loadCodeAssist`, falls back to `GOOGLE_CLOUD_PROJECT` when Code Assist discovery fails, and supports bundled Gemini CLI credential extraction from `bundle/*.js` layouts.

## Evidence
