## Summary

Current `main` already fixes the stale plugin-HTTP-registry drift that could leave dynamically registered webhook routes unreachable. The fix is present in the latest release commit `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5` (`v2026.4.22`), and regression tests now cover the exact stale-registry 404 shape.
