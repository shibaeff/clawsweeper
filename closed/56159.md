## Summary

Current `main` already routes heartbeat runs through the normal agent execution path and now has explicit heartbeat-only bootstrap/context handling with regression tests. The main proof is commit `028b6c9b13c960b6f59a7643aff4f6de04e07f71`, which adds heartbeat-specific bootstrap preservation and prevents continuation-skip from bypassing heartbeat context setup.
