## Summary

Current main already uses the curl/PowerShell installer as the documented primary path, defaults to npm rather than Homebrew, offers direct npm/pnpm/bun installs, and documents a local-prefix `install-cli.sh` flow specifically for no-root environments. On Linux, the installer code uses distro package managers or NodeSource, not Homebrew.
