## Summary

Current `main` already blocks `agent`, `chat`, and `chat.side_result` broadcasts from non-operator clients through the gateway scope table, and it includes regression coverage proving node-role clients do not receive those events. The same implementation is present in shipped release `v2026.4.23`, so this PR is redundant.

## Evidence
