## Summary

Current main no longer relies on the old pi-ai Anthropic provider path for default Anthropic traffic. It routes Anthropic models through OpenClaw's boundary-aware `anthropic-messages` transport, resolves missing reasoning to disabled thinking, and maps Claude 4.6 reasoning to adaptive thinking with explicit effort. The current docs also document Claude 4.6 adaptive defaults and supported config.

## Evidence
