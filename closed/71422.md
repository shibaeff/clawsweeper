## Summary

Keep open. Current `main` only partially addresses the regression: the Control UI now blobifies authenticated local avatar fetches and the chat renderer accepts `blob:` URLs, but the shipped Control UI CSP still blocks `blob:` images and its test suite explicitly enforces that restriction. That matches the reporter’s follow-up that blobification succeeded but rendering still fell back until CSP was patched locally.
