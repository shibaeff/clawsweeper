## Summary

Current main already implements the env-proxy bootstrap for CLI commands, including gateway startup, using Undici's global dispatcher rather than patching `globalThis.fetch`, so this PR is redundant on `main`.

## Evidence

- **Current main confirmed:** Checked out HEAD matches the provided current main SHA `7ac35b4f69009490180a35bf404638d4280a4865`.
  - command: `git rev-parse HEAD`
  - sha: [7ac35b4f6900](https://github.com/openclaw/openclaw/commit/7ac35b4f69009490180a35bf404638d4280a4865)
- **CLI bootstraps env proxy before command routing:** `runCli` loads env, initializes proxy capture, then calls `ensureGlobalUndiciEnvProxyDispatcher()` before dispatching CLI work. That means `openclaw gateway ...` already inherits env-proxy handling without a gateway-local `fetch` patch.
  - file: [src/cli/run-main.ts:178](https://github.com/openclaw/openclaw/blob/7ac35b4f69009490180a35bf404638d4280a4865/src/cli/run-main.ts#L178)
- **Main has the dispatcher-based implementation:** `ensureGlobalUndiciEnvProxyDispatcher()` installs `new EnvHttpProxyAgent()` with `setGlobalDispatcher(...)` when HTTP/HTTPS proxy env is configured, and avoids clobbering unsupported custom proxy dispatchers.
  - file: [src/infra/net/undici-global-dispatcher.ts:91](https://github.com/openclaw/openclaw/blob/7ac35b4f69009490180a35bf404638d4280a4865/src/infra/net/undici-global-dispatcher.ts#L91)
- **Timeout composition is preserved:** `ensureGlobalUndiciStreamTimeouts()` detects the active dispatcher kind and reapplies timeout tuning on top of an `EnvHttpProxyAgent`, addressing the earlier review concern that a direct `globalThis.fetch` override would bypass later dispatcher hardening.
  - file: [src/infra/net/undici-global-dispatcher.ts:118](https://github.com/openclaw/openclaw/blob/7ac35b4f69009490180a35bf404638d4280a4865/src/infra/net/undici-global-dispatcher.ts#L118)
- **Behavior is covered by tests:** The current test suite covers env-proxy install, idempotence, retry after unsupported dispatchers, and reinstall after external resets.
  - file: [src/infra/net/undici-global-dispatcher.test.ts:207](https://github.com/openclaw/openclaw/blob/7ac35b4f69009490180a35bf404638d4280a4865/src/infra/net/undici-global-dispatcher.test.ts#L207)
- **Proxy bootstrap is also used in the embedded Pi attempt path:** The embedded attempt runtime explicitly calls `ensureGlobalUndiciEnvProxyDispatcher()` before timeout tuning, which is the concrete path discussed in this PR body.
  - file: [src/agents/pi-embedded-runner/run/attempt-http-runtime.ts:7](https://github.com/openclaw/openclaw/blob/7ac35b4f69009490180a35bf404638d4280a4865/src/agents/pi-embedded-runner/run/attempt-http-runtime.ts#L7)
