## Summary

Keep open. Current `main` has a MiniMax speech provider, but it is still API-key-only (`minimax`), while the OAuth `minimax-portal` path explicitly excludes speech. The requested Token Plan OAuth TTS path and the requested `speech-02-hd` family are not implemented on current `main`.

## Evidence
