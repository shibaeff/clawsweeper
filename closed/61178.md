## Summary

Close as `clawhub`. Current `main` is explicitly plugin-first for this class of feature: third-party skills/plugins are meant to ship outside core via ClawHub/npm, and OpenClaw already exposes a context-engine plugin seam that runs before each model prompt. The linked SkillPilot package fits that external plugin/context-engine path better than a built-in core integration.

## Evidence
