## Summary

Keep open. Current `main` still passes configured Google music/video `baseUrl` values through `normalizeGoogleApiBaseUrl`, while the shared helper `resolveGoogleGenerativeAiApiOrigin` explicitly strips a trailing `/v1beta`. The PR's fix and added regression coverage are not present on `main`, so this is still plausible unresolved work rather than cleanup-close material.
