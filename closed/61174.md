## Summary

Close as plugin-scoped: the report targets `@tencent-weixin/openclaw-weixin` outbound/context-token behavior, and current `main` explicitly documents WeChat transport, iLink API calls, and context-token ownership as external plugin responsibility rather than OpenClaw core.
