## Summary

Current `main` already covers the reported cron fallback problem: the Anthropic plugin now recognizes `claude-haiku-4-5` as a modern Claude ref, and cron model-resolution fallback now emits an explicit warning instead of failing silently. The same code is present in release `v2026.4.23`.
