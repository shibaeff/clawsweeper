## Summary

Current main already addresses the reported false-positive path. Local authenticated gateway probes are now kept device-bound so read/detail RPCs do not get scope-limited, and the shipped status/probe surfaces classify true missing `operator.read` cases as degraded reachability instead of a hard failure.
