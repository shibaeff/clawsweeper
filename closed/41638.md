## Summary

Current `main` already gives the cron edit panel independent scrolling. `.cron-workspace-form` has a constrained `max-height` plus `overflow-y: auto` in the base rule, and the narrow-screen media query only changes `position` and `order`, so this PR's extra mobile override is redundant and less precise than the shipped `calc(...)` height.
