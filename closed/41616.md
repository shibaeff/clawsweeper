## Summary

Current `main` already ships the headed-browser Docker workflow this PR is trying to add, via the maintained sandbox-browser path: `Dockerfile.sandbox-browser`, `scripts/sandbox-browser-entrypoint.sh`, runtime wiring in `src/agents/sandbox/browser.ts`, published docs, and regression tests. The latest release `v2026.4.23` contains that implementation, so this PR’s separate Ubuntu+Chrome gateway wrapper is redundant and weaker than the supported core path.
