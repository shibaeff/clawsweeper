## Summary

Current `main` already includes an Ollama implicit-discovery/runtime-registration path that covers the reported failure mode: the bundled Ollama plugin now discovers local models without an explicit `models.providers.ollama` entry, preserves the `ollama-local` marker through runtime auth resolution, and falls back to lightweight discovery-provider synthetic auth when the full runtime hook is not yet loaded. The implementing commit is on `main` but not in a tagged release yet.

## Evidence
