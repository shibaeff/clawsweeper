## Summary

Current `main` already routes Telegram native slash commands through topic-qualified session targets and preserves the topic-qualified origin target. The latest release tag `v2026.4.22` contains the same implementation and regression tests, so this issue is already fixed on shipped code newer than the reported `2026.3.24` version.

## Evidence
