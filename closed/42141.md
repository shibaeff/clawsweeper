## Summary

Current `main` already implements DuckDuckGo web search support through the bundled plugin architecture, including runtime selection, keyless fallback behavior, onboarding/configure coverage, docs, and tests. The feature was already present in release commit `00bd2cf7a376f1fba26291c6c4766f1f15cbdfa5` (`v2026.4.22`), so this PR is redundant.
