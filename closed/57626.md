## Summary

Current main already has the reported safeguards: embedded runs now abort stalled model streams via the LLM idle-timeout path, compaction safeguard writes a boundary instead of re-trigger looping on empty/internal-only history, and silent `NO_REPLY` assistant entries are filtered from outbound/chat-history surfaces. The changelog shows these protections shipped after 2026.3.28 and are present in the current code.
