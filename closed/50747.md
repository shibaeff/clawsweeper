## Summary

Current `main` already implements the parent/subagent completion path and contains regression coverage for the reported stuck-queued behavior: `sessions_yield` now exits cleanly so the parent session becomes idle, and the subagent lifecycle registry consumes completion events and triggers announce/cleanup back to the requester.

## Evidence
